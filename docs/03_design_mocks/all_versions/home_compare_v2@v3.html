🤔 A vs B の選び方

  • A 推奨派の論点: スクロール 2 回で全部見える、毎日触るホームは軽い方が良い、コース選びは「コース」タブで完結
  • B 推奨派の論点: 初回ログイン直後やコース選び迷子時に強い、進捗バー付きで「次やる候補」が一望できる
  • 折衷案: A をベース + 入会 7 日間だけ B モード（オンボーディング期間中はコース一覧強調 → 慣れたら A に自動切替）

⚠️ 未確定 / 要相談

  • 機能ブロックの 4 つ目「マイページ」は下部タブの「設定」と機能が一部重複（プロフィール表示）。マイページ = プロフィール表示 / 設定 = アプリ設定と分離するか、統合するか
  • 機能ブロックの数（4 vs 6）: 「目標シート」「FAQ/ヘルプ」を追加する選択肢もある
  • キャラクター（添付トレーナー）の登場箇所: 起動ローディング / 章末祝福 / 空状態 のどこに置くか
  • 絵文字 → カートゥーンアイコンへの差し替え素材調達
