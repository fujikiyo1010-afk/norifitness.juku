🗺 この画面の役割

  • 履歴一覧 (ブロック C) で 1 月分タップした後に開く画面
  • 過去の自分の回答 + のりfitness返信 をワンスクロールで見れる "アーカイブ単票"

📐 構成 (上から)

  • ヒーロー帯: 月 + 送信日 + 返信日 (E案グラデ)
  • のりfitness返信: 動画 (大) + コメント (薄黄背景で目立つ)
  • 全体スコア: 平均 / 合計 / 前月比 の 3 数値
  • 提出した内容: 17 項目をカテゴリ別に、Q + スコア で羅列
  • CTA: 「次の月次添削に参考」「履歴に戻る」

🤔 そもそも この画面、要る? — 議論

○ 追加する利点
・受講生のシーン B「ふと過去を振り返る」を満たす
・のり返信動画を月別に整理して再アクセスできる
・次の月次添削を書く時、前月を参照できる (シーン C)
・「のりfitnessコメントライブラリ」としての価値が積み上がる
× 追加する負担
・画面数が 3/3 → 4/3 (実質) に増える
・実装コスト上がる
・履歴一覧で全部済むなら不要かも
・受講生が実際に見るかは未知数 (使われない可能性)

💡 代替案

  • 案 α: 詳細画面ナシ — 履歴一覧の行をタップしても何も起きない (or 動画だけモーダル展開)
  • 案 β: 詳細画面アリ (今回モック) — 1 月分の単票画面に遷移
  • 案 γ: アコーディオン展開 — 履歴一覧の行タップで、その場で下に展開 (画面遷移なし)

🗳 きよむさんに判断してほしいこと

  • このモック (案 β) で「実装する価値あり」と感じるか?
  • もし重いなら、案 γ (アコーディオン) でも代用できそうか?

  • 17 項目の質問文 = 仮 (実際の項目定義は別途確定要)
