🗺 構成 (上から)

  • ヘッダー帯: 「今日の記録」+ 日付 (温かグラデ E 案)
  • 入力カード: 体重 / ウエスト / 首回り (3 項目)、体脂肪率は自動計算
  • 記録するボタン: ティール緑、影付き
  • 自動反映の案内: 「目標管理シート ① と 変化を見る に自動反映」
  • 最近の記録: 過去 7 日のリスト、未記録日も表示
  • 遷移リンク 2 つ: 変化を見る / 目標管理シート
  • 下部 5 タブ: 記録 active

📝 入力 UI のポイント

  • 3 項目だけ入力: 体重・ウエスト・首回り (毎日変動する数値)
  • 身長は除外: プロフィール連動、毎日入力させない (変わらない)
  • 体脂肪率は自動計算: アメリカ海軍式 (性別自動切替) で入力 3 つから算出
  • 大きめのフォントサイズ 16px: タップしやすい、数値を打ちやすい
  • 右寄せ + 太字: 数値感を強調

🔗 連動の流れ

① 受講生がこの画面で 今日の体組成を記録
② body_metrics テーブルに保存
③ 目標管理シート ① 現状を把握 = body_metrics の最新行を表示
④ 📈 変化を見る 画面で時系列グラフ描画

✅ 反映した合意事項

  • 責めない方針: 未記録日も表示するが「未記録」とだけ書く、催促しない
  • Streak 抑制: 「3 日連続」みたいな表記なし
  • 習慣化のリマインド: 「無理なく続けましょう」だけ表示
  • 自動連動: 目標シート ① と 変化を見る への明示

🌐 ホームからの誘導 (実装時)

  • ホーム画面の進捗バー下に「📊 今日の体組成を記録する →」リンクを追加
  • 記録タブ (マイ学習記録) の中にも「📊 体組成記録」カードを追加
  • 両方からこの画面にアクセス可能

⚠️ 検討余地

  • 未記録日の扱い: 「未記録」表示で OK? それとも非表示?
  • 記録時間帯のヒント (「朝起きてすぐ」) の必要性
  • 記録完了時のフィードバック (Toast「✓ 記録しました」)
  • 体重が前日より増えた時の表示 (現在 赤 で「+0.2」、責めないなら緑色 or 表示なし?)
