## 2. 主要ページの動線詳細

### A. 認知 → 申込

```
TVCM / Web広告 / YouTube
   ↓
公式LP（rizap.jp）
   ↓
[無料カウンセリング予約フォーム]
- 名前、電話番号、メールアドレス
- 希望店舗 / 希望日時
   ↓
予約完了メール（電話確認あり）
   ↓
店舗訪問（90分のカウンセリング）
```

### B. カウンセリング → 契約

```
店舗到着
   ↓
[1] ヒアリング（目標・現状・生活習慣）30分
   ↓
[2] 体組成測定（InBody等）10分
   ↓
[3] トレーニング体験（任意）20分
   ↓
[4] プラン提案・料金提示 20分
   ↓
[5] 契約手続き（クレカ・分割）10分
   ↓
契約完了 → 初回トレーニング日設定
```

### C. 契約後の体験

```
RIZAP touch アプリインストール
   ↓
専属トレーナー初対面 + プラン共有
   ↓
週2回トレーニング（マンツーマン、店舗）
   ↓
毎日: 食事撮影 → アプリ送信
   ↓
トレーナーから24h以内に返信
   ↓
週次: 体重・体脂肪率測定（店舗）
   ↓
2ヶ月後: 卒業・アフター案内
```

## 3. 入会から退会までのフルジャーニー

| フェーズ | 期間 | 顧客の状態 | 仕掛け |
|---|---|---|---|
| 認知 | - | CM・広告経由で「結果にコミット」を知る | TVCMの圧倒的露出 |
| 興味 | - | LPでビフォーアフター閲覧 | 写真の説得力 |
| 検討 | 数日〜数週間 | 無料カウンセリング予約 | 「無料」で心理ハードル↓ |
| 体験 | カウンセリング当日 | 店舗で生身の対応・体組成測定 | プロのヒアリング |
| 決断 | カウンセリング後半 | 価格提示・分割提案 | 60回分割で月額化 |
| 契約 | 当日〜数日 | クレカ・分割で契約 | 30日返金保証で安心 |
| 開始 | 初回トレーニング | 専属トレーナーと対面 | 期待値最大 |
| 継続 | 2ヶ月（16回） | 週2トレ＋毎日食事報告 | アプリで毎日接触 |
| 達成 | 終了直前 | 結果が出る | 写真撮影・実績化 |
| 卒業 | 2ヶ月後 | アフター提案 | PRIME プラン継続誘導 |
| 退会 | 任意 | 解約 or 完了 | 30日返金 / 解約時の細則あり |

## 6. 筋肉塾新サイト設計に活かせるポイント ⭐

### ① 「結果にコミット」型の明快なメッセージ
- ビフォーアフター写真の徹底活用
- 数字（体重・体脂肪率の変化）を前面
- **筋肉塾**: のり氏自身や卒業生のビフォーアフター + 数字訴求

### ② 30日返金保証 = 高額商品の購買障壁を下げる
- 30日間の全額返金（販売物品除く）
- これがあるから40万円近い金額を出せる
- **筋肉塾**: ¥327,800 の購入ハードルを下げるため、何らかの返金保証 or 「最初の◯回で成果が出なければ返金」型を検討

### ③ 60回まで分割可能（月額化）
- 頭金5,000円〜・月額数千円台に分解
- 心理的に「月◯円なら払える」と感じさせる
- **筋肉塾**: 分割払い導入を検討（10〜24回程度）

### ④ アプリ「RIZAP touch」の食事写真フロー
- 受講生は撮影 → 送信だけ（テキスト入力不要）
- トレーナーが返信で毎日接触
- **筋肉塾**: 食事写真のアップロード機能は **MVP の必須要件候補**。タイムライン式 UI も検討

### ⑤ 専属トレーナー制
- マンツーマンで担当固定
- 「あなたのトレーナー」というオーナーシップ感
- **筋肉塾**: のり氏 or 認定講師との 1on1 機能を持つかどうかの分岐点

### ⑥ 無料カウンセリングの全国均一化
- 全店舗で同じプロトコル
- カウンセリング = セールスの場
- **筋肉塾**: オンライン無料相談を入会前提に組み込む可能性

## 7. 避けるべきアンチパターン

### ❌ 店舗依存・オンライン完結不可
- 物理店舗訪問が必須 = 商圏が限定
- **筋肉塾は完全オンラインの強み**を活かし、この逆を行く
- ライザップが「オンライン完結版」をうまく作れていない事実は **強い示唆**

### ❌ 高額な印象を払拭するための広告依存
- TVCM の露出量で「みんな知ってる安心感」を作る
- 中小事業者には真似できない戦略
- **筋肉塾は「のり氏のYouTube・SNS」が代替メディア**になる

### ❌ 営業色の強いカウンセリング
- カウンセリング = 高確率で契約させる場
- 「営業されに行く」と感じる層が一定数いる
- **筋肉塾はオンライン相談 = 教育＋ローンチ型販売**で柔らかく代替できる

### ❌ アプリの体験設計が「報告 → 返信」のシンプル型に留まる
- ゲーミフィケーション・コミュニティ要素なし（推測）
- 受講生間の交流がない
- **筋肉塾は同期コミュニティ・進捗共有機能で差別化可能**
