招待コードでアカウント作成

招待コードを入力

LINE で受け取った 12 桁の招待コードを
ハイフン込みで入力してください

例: ABCD-1234-WXYZ (12 文字 + ハイフン 2 つ)
すでにアカウントをお持ちの方 → ログインへ

画面の役割

  • のり氏が手動でアカウント発行 → LINE で招待コード送付 → 受講生がこの画面で入力
  • 正しい招待コード入力 → メアド + パスワード設定画面 (or オンボーディング) へ進む

構成 (上から)

  • ヘッダー: ← 戻る + 「招待コードでアカウント作成」
  • アイコン (キーパッド) + 「招待コードを入力」 + 説明文
  • コード入力欄 (大きめ・中央寄せ・Inter フォントで読みやすく)
  • 「次へ →」 ボタン
  • 「ログインへ」リンク (既存アカウント持ち向け)

動線

  • 正しいコード → アカウント情報入力画面 (メアド + パスワード)
  • 不正コード → エラーメッセージ表示 (このフォーム内で)
  • ログインへ → /login
