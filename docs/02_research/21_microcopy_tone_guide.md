# 21 マイクロコピー & トーンガイド

**作成日**: 2026-05-20
**目的**: 新サイト全体の **言葉遣い**を「のり氏らしさ」「学習ツール最優先」「ゲーム感排除」の3軸で統一する。
**根拠**:
- [CLAUDE.md](../../../CLAUDE.md) の文体ルール（みなさん / あなた、絵文字控えめ等）
- [20_norisan_brand_analysis.md](20_norisan_brand_analysis.md) の既存LPコピー分析
- [07_wireframe_members.md](07_wireframe_members.md) v2.1 の各画面に必要な文言

## 🎯 3つの原則

### 原則1. 「みなさん」「あなた」を文脈で使い分ける
- **「みなさん」**: 全体への語りかけ、講師から受講生全体へ
- **「あなた」**: 個別の操作画面、一対一の語りかけ
  - 例: 「あなたの目標を聞かせてください」 「みなさんで頑張りましょう」

### 原則2. 「のり氏らしさ」=「真摯・温かい・押し付けない」
- ❌ 「絶対に」「必ず」「べきです」 → 強要・押し付け感
- ✅ 「やってみましょう」「一緒に」「無理せず」 → 寄り添い感
- ✅ 実績・科学エビデンスは数字で示す（「1,000人以上の指導」型）

### 原則3. 学習ツール最優先 / ゲーム感排除
- ❌ 「XP獲得！」「Lv UP!!」「コンボ！」 → 軽薄
- ✅ 「累計 48回」「章2マスター達成」 → 学びの言葉
- ❌ 「やらないと罰」「Streak ロスト！」 → 罰則感
- ✅ 「明日からまた一緒に」「無理せず続けましょう」 → 寛容

## 🎙 コピーの「型」5パターン

のり氏 LP から抽出した型を、新サイトで使えるテンプレートに翻訳。

### 型1. 「数字 + 信頼」型
- **既存例**: 「1,000人以上の指導で見えてきた " 本当の選び方 "」
- **新サイト適用**:
  - 「卒業生 200名以上が達成」
  - 「平均 3ヶ月で 5kg 達成」

### 型2. 「あなた呼びかけ + 共感」型
- **既存例**: 「リンゴ酢は " ただ飲むだけ " では効果半減」
- **新サイト適用**:
  - 「動画を " 見るだけ " では身につきません。3行で振り返ってみましょう」
  - 「目標は " 決めるだけ " では達成できない。今週の行動を1つ決めてみましょう」

### 型3. 「努力を讃える」型
- **既存例**: 「やめなかった人だけが」
- **新サイト適用**:
  - 「ここまで続けてきた、あなたへ」（卒業画面）
  - 「8週続けてくれて、ありがとう」（バッジ獲得時）

### 型4. 「" 括弧 " 強調」型
- **既存例**: 「" 本当の "」「" ただ飲むだけ "」
- **新サイト適用**: キーフレーズを `" "` で囲むことで強調
  - 「" できた！" を押すと記録されます」
  - 「" 自分の言葉 " で書くことが、定着への近道」

### 型5. 「やわらかな促し」型
- **既存例**: 「やってみましょう」「一緒に」
- **新サイト適用**:
  - 「今日の学びを書いてみましょう」（強要じゃなく、提案）
  - 「次の章へ進んでみましょう」

## 🚫 全面禁止のコピー

絶対に使ってはいけない表現:

| カテゴリ | NG ワード | 理由 |
|---|---|---|
| ゲーム感 | XP, Lv, EXP, レベル, ランク, ボーナス, コンボ, ストリーク | のり氏ブランドとズレ |
| 罰則感 | 失う, 罰金, ペナルティ, 強制, やらないと | 学習ツール最優先 / 寛容 |
| 押し売り | 必ず, 絶対に, 今すぐ, 限定◯名, 残り◯個 | 教育内コンテンツに不要 |
| 強要 | 〜してください, 〜しなければなりません | 寄り添い感を損なう |
| カタカナ多用 | サブミット, キャンセル, アップロード, ダウンロード | 平易な日本語に置き換え |
| 英語 | Done, Save, Cancel, OK, Submit | 全部日本語化 |

→ ただし **「ログイン」「メール」「パスワード」** など定着した語は許容

## 🟢 タイトル・見出しのコピー禁則

### 禁止
- 「**◯◯機能**」 ← 「機能」呼ばわりは冷たい
  - ❌ 「コメント機能」 → ✅ 「💬 コメント」
  - ❌ 「目標管理シート機能」 → ✅ 「🎯 私の目標」
- 「**◯◯一覧**」 ← 業務システム感
  - ❌ 「ブックマーク一覧」 → ✅ 「🔖 動画ブックマーク (12件)」

### 推奨
- **絵文字 + 短い名詞句**（見出しに1個程度）
- **「私の」「今日の」「あなたの」** で所有感
- **件数表示は () で添える** （例: `(12件)`、 `(5/12)`）

## 🔠 英語・カタカナ vs 日本語 ルール

### 全て日本語化（強制）
- Save → 保存
- Cancel → キャンセル ❌ → 「やめる」「閉じる」 ✅
- OK → 「確認」「閉じる」
- Done → 「完了」「✓」
- Loading → 「読み込み中...」
- Error → 「エラー」 もしくは具体的な日本語
- Login → ログイン（定着語、許容）
- Submit → 「送信」

### カタカナ化された業界用語（許容範囲）
- パスワード、メール、アカウント、プロフィール、ログイン、ログアウト、ホーム、コメント、ブックマーク、サポート、ヘルプ、FAQ
- ※ 定着しすぎてて代替が不自然なもののみ

## 📦 トーン・チェックリスト（実装時）

新規コピーを書く時、以下5項目をチェック:

- [ ] **押し付けてないか?** 「〜してください」より「〜してみましょう」
- [ ] **ゲーム用語ないか?** XP, Lv, ランク, ストリーク等
- [ ] **絵文字は見出し1個程度に収まっているか?** 本文連発NG
- [ ] **英語のままになってないか?** Save / Cancel / Done 等
- [ ] **数字や実績で説得力を増せないか?** 「1,000人以上」型
