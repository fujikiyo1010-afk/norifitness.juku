# 旧筋肉塾 vs 新装 ・ 機能比較資料

**作成**: 2026-06-22
**対象**: ソフトローンチ参加社員 向け説明資料
**目的**: 「旧サイポンと比べて何が良くなったか / 何ができるようになったか」 を共有

---

## 📊 サマリ ・ 1 行で

```
旧 = 「動画見て終わり」 の 一方向 学習
新 = 「動画 + 個別指導 + 双方向 + 自走化」 の 完全プラットフォーム
```

---

## 🎓 旧筋肉塾 (= サイポン) の現状

| 項目 | サイポン |
|---|---|
| **コンテンツ** | 5 コース / 22 章 / 217 レッスン |
| **動画視聴** | サイポン埋め込み (= モバイル最適化弱) |
| **進捗管理** | 視聴チェックのみ / 続きから戻る ◎ ない |
| **個別最適化** | 全員同じコンテンツ / メニュー個別なし |
| **サポート** | LINE 個別 / スレッド管理なし |
| **月次添削** | **無し** |
| **プロテイン発送** | 手動管理 |
| **入会** | LP → 申込 → 手動でアカウント発行 |
| **リマインド** | **無し** (= 離脱しても気付かない) |
| **管理画面** | 簡素 / 一覧管理が手間 |

---

## 🆕 新装で **改善** したこと (= 全 50+ 項目)

### 🎓 学習体験 (= 5 項目)

| 改善 | 詳細 |
|---|---|
| **続きから学ぶ** | 前回最後に視聴したレッスンに 1 タップで戻れる |
| **完了履歴** | 視聴済みレッスンを記録 / 進捗 % 可視化 |
| **2 階層構造** | コース → 章 (= アコーディオン) → レッスン / 迷わない |
| **Vimeo 完全埋め込み** | スマホ最適化 / 倍速再生 / 字幕対応 / iPhone 100% 動く |
| **学習リマインド** | 3 日視聴なし → Push で学習タブ直接案内 |

### 🩺 個別最適化 (= 4 項目)

| 改善 | 詳細 |
|---|---|
| **筋トレカルテ 9 問** | 性別 / 環境 / 頻度 / 重点部位 / 経験 / けが 等 構造化 |
| **メニュー個別配布** | 277 件 過去事例 × 受講生カルテで類似度マッチング → admin 配布 |
| **カルテ更新リクエスト** | 受講生が「状況変わったので変更してほしい」 を送る動線 |
| **メニュー変更リクエスト** | 受講生が「腹筋追加して」 等を送る動線 |

### 📊 自己モニタリング (= 4 項目)

| 改善 | 詳細 |
|---|---|
| **目標管理シート** | 5 セクション 17 項目 + 8 セルフイメージ / 自己分析 |
| **目標シート添削** | のり氏が項目ごと + セクションコメント / 黄色ハイライト表示 |
| **体組成記録** | 体重 / 体脂肪率 / ウエスト / 首回り / 身長 + 任意メモ |
| **月次添削 17 項目** | 自己評価 → のり氏 動画返信 (= Vimeo upload) |

### 📨 双方向コミュニケーション (= 7 項目)

| 改善 | 詳細 |
|---|---|
| **アプリ内チャット** | スレッド管理 / リアルタイム / 既読 |
| **Push 通知 13 種類** | メニュー配布 / 添削完了 / 動画公開 / リクエスト返信 / リマインド 等 |
| **メール通知 11 種類** | Welcome / 添削完了 / 発送 / アナウンス 等 |
| **一斉アナウンス** | 全受講生に 規約改定 / メンテ告知 / インシデント連絡 |
| **リマインド cron 5 種類** | カルテ 5 日 / 目標シート 7 日 / 体組成 7 日 / 学習 3 日 / 月次添削 月末 |
| **3 段階リマインド** | 各リマインド 7 日 cooldown / 受講生が動けば自動停止 |
| **opt-out 不可 メール 2 種類** | Welcome / 申請受領 (= 重要メールは必ず届く) |

### 🛒 運用効率化 (= admin 側) (= 8 項目)

| 改善 | 詳細 |
|---|---|
| **マジックリンク入会** | 手動アカウント発行 → admin が 招待 1 クリック → メール自動 |
| **オンボーディング 8 ステップ** | 入会直後の受講生を強制的に正しい動線へ (= プロテイン発送先必須) |
| **オンボ Step 0 PWA 強制** | ホーム画面追加しないと進めない / アプリ感保証 |
| **iOS Safari / Android Chrome 判定** | Chrome で開いた人に「Safari に切り替えて」 自動案内 |
| **プロテイン発送管理** | 一覧で「未発送 N 件」 / 2 日経過で admin にアラートメール |
| **/admin ダッシュボード** | 月次完了率 / 未対応件数 / 重要アラート 1 目で |
| **受講生一覧 8 カラム** | カルテ / 目標シート / 月次 / メニュー / リクエスト 状態 = ハブ開かなくても全把握 |
| **受講生 個別ハブ** | 1 画面で その人の全データ / 全アクション可能 |

### 🩹 リクエスト対応動線 (= 3 項目)

| 改善 | 詳細 |
|---|---|
| **リクエスト → 編集 → 返信 自動連携** | カルテ更新リクエスト → 「カルテを編集」 → 「保存して返信フォームへ」 → テンプレ挿入 → 「送信 + 対応済」 まで 1 動線 |
| **テンプレ機能** | 返信文をテンプレ管理 (= 「ご要望反映しました」 等) |
| **黄色 リクエスト処理中バナー** | admin が処理中なのが視覚で分かる |

### 📱 アプリ化 (= PWA) (= 5 項目)

| 改善 | 詳細 |
|---|---|
| **ホーム画面追加** | アプリらしい起動 / 全画面表示 |
| **iOS Push 通知** | iOS 16.4 + で PWA でも Push 受信可能 |
| **オフライン対応** | 圏外で 「再接続を試す」 専用画面 |
| **スプラッシュ画面** | 起動時の白画面なし (= ティール緑 + アイコン) |
| **キャッシュ自動更新** | Push 着地時 / バックグラウンドから戻り時に最新表示保証 |

### 🎨 UX 細部 (= 8 項目)

| 改善 | 詳細 |
|---|---|
| **ローディングスピナー統一** | 主要 13 ボタンで「送信中…」 + スピナー表示 |
| **ページ遷移プログレスバー** | 画面上部に細いティール緑線 (= 200ms 以上だけ表示) |
| **強制ライトモード** | ダークモード時の中途半端な配色崩れを防止 |
| **画面下のボタンが隠れない** | h-[100dvh] + 下部ナビ非表示 で iPhone 最適化 |
| **戻るボタン 1 個** | 内側ヘッダー重複削除 / 受講生が迷わない |
| **郵便番号 7 桁バリデーション** | 不完全な郵便番号で進めない |
| **送信ボタン位置** | 最後の入力欄の真下 / スクロールで隠れない |
| **Welcome メール簡潔化** | 「まずホーム追加」 + 「あとはアプリの案内に沿って」 のみ / 二重指示なし |

### 🔒 セキュリティ + インフラ (= 6 項目)

| 改善 | 詳細 |
|---|---|
| **RLS 全 40 テーブル** | 受講生は自分のデータのみ閲覧 / 他人見えない |
| **rate limit** | ログイン失敗 10 回/5 分で制限 |
| **Resend ドメイン認証** | SPF / DKIM / DMARC 通過 / 迷惑メール扱いされにくい |
| **HTTPS + Cloudflare** | 全通信暗号化 + CDN 高速化 |
| **Service Worker キャッシュ戦略** | navigate / static / api 別キャッシュ |
| **オンボーディング ガード** | 未完了受講生がホーム直行する抜け穴塞済 |

### 📋 ホームバナー (= 5 項目)

ホーム画面の上部に「やってない事」 を黄バナーで強制誘導:
- カルテ未記入
- 目標シート未記入
- 体組成 未記録 / 途絶 (= 7 日)
- 通知 OFF (= ON 推奨)
- → 各バナータップで対象画面に直行

## ⏳ まだ **改善余地** がある (= 線② / Phase 4 候補)

これらはソフトローンチ後の継続改善対象です:

### 受講生 機能拡張

| 項目 | 内容 |
|---|---|
| **体組成推移グラフ** | 体重 / 体脂肪率 等の時系列可視化 |
| **編集履歴** | 目標シートの過去スナップショット閲覧 |
| **過去の添削履歴一覧** | 月次添削 過去動画 一覧 |
| **学習動画 ブックマーク** | 後で見直したいレッスン管理 |
| **実践リスト** | 「やってみる」 メモ |
| **学習動画 完了履歴フィルタ** | 視聴完了済みのみ表示 |
| **試験合格画面** | コース末尾の試験 + 合格演出 |
| **コース修了証** | 全コース完走時のお祝い |

### 通知 / 連携機能

| 項目 | 内容 |
|---|---|
| **LINE 連携** | 通知を LINE で受け取る選択肢 |
| **個別通知 ON/OFF** | 「メニュー配布だけ ON、 月次添削 OFF」 等タイプ別 |
| **お知らせ機能** | 管理者から受講生への お知らせカード |

### admin 効率化

| 項目 | 内容 |
|---|---|
| **admin 通知設定 UI** | リマインド日数を画面から変更 |
| **admin 検索フィルタ拡充** | 受講生一覧で カルテ未提出だけ 等 |
| **admin daily まとめメール** | 毎朝「未対応件数」 を まとめて 1 メール |

### UX / デザイン

| 項目 | 内容 |
|---|---|
| **スケルトンスクリーン** | データ取得中の骨組み表示 |
| **動画プリロード** | 月次添削動画の体感速度向上 |
| **管理画面 デザイン統一** | 全面リブランド (= 色 / 余白 / フォント) |
| **PWA アイコン リブランド** | 「ティール緑塗り + キャラ中央」 統一版 |
| **キャラの追加** | 様々な表情のキャラ / 演出 |

### システム

| 項目 | 内容 |
|---|---|
| **Sentry SDK 修復** | Next.js 16 + Turbopack 互換性 SDK 更新待ち |
| **試験 RLS 強化** | correct_choice_id の露見抑制 |
| **管理者 PWA** | きよむさん用 Mac Safari admin PWA |
