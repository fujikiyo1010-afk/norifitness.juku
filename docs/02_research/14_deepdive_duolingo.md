# 14 深掘り: Duolingo

**URL**: https://www.duolingo.com/
**料金**: 無料（広告付き） / Super Duolingo 約1,300円/月
**販売形態**: アプリ + Web、フリーミアム
**筋肉塾との関係**: **継続率・完工率の世界一**、ゲーミフィケーション心理設計の教科書 ⭐⭐⭐

---

## 1. サービス概要

| 項目 | 内容 |
|---|---|
| 概要 | 世界最大の語学学習アプリ。月間アクティブユーザー 1億人超 |
| 主な対象 | 全世代、特に「習慣化が苦手な学習者」 |
| 強み | **継続率設計の極致**、「やめさせない」心理メカニズムの完成形 |

---

## 2. 主要な継続メカニズム（5層構造）

### 層1: Streak（連続記録）
- 毎日1レッスン完了で +1日
- アプリ全画面に「現在の連続日数」表示
- **損失回避心理**: 「ここまで続けた記録を失いたくない」
- **データ**: 7日 Streak 達成者は次日復帰率 2.4倍、コース完了率 3.6倍

### 層2: XP（経験値）
- レッスン完了で XP 獲得
- すべての行動が XP に集約される単一指標
- Streak / League / Achievement の全てに紐付く

### 層3: League（リーグ）
- **Bronze → Silver → Gold → Sapphire → Ruby → Emerald → Amethyst → Pearl → Obsidian → Diamond**
- 週単位でリーグ内のXP獲得量で順位決定、上位昇格・下位降格
- 同じレベル帯と競争 = **負けすぎず、勝ちすぎず**
- **データ**: リーグ導入で完了率 +25%、XP リーダーボード閲覧者は完了レッスン数 +40%

### 層4: Achievement（実績バッジ）
- 「初心者卒業」「7日連続」「100問正解」等の達成バッジ
- 階段状に難易度上がる
- レアバッジで長期目標

### 層5: Friend Streaks / Social
- 友達と連動した Streak
- 社会的責任で「相手にも迷惑かけたくない」心理

---

## 3. 学習継続を促す細やかな仕組み

### Daily Goal（個人化された日次目標）
- ユーザーが「Casual（5分）」〜「Insane（20分）」を自分で選ぶ
- **重要な発見**: 「目標を高く設定する人ほど Streak を失いやすい」
- → Daily Goal と Streak を **分離**し、Streak は「1レッスンでも OK」に
- 「**ハードル下げる柔軟性 × Streak の絶対性**」のバランス

### Streak Freeze（連続記録の凍結）
- 1日忘れても Streak 維持できる救済アイテム
- **罰則的でない・寛容な設計** = 「燃え尽き」回避

### Built-in Breaks（休憩促進）
- 「今日は休みましょう」と通知してくる日がある
- 強制じゃない、優しい設計

### Notification（通知の心理設計）
- 「フクロウ Duo」キャラが「待ってます…」と訴える
- ユーモラス・自虐的で、押し付けがましくない
- 通知タイミングはユーザーの過去アクセス時刻を学習

### Hearts（ハート / 体力制）
- 間違えると Heart 減る、ゼロで一時停止
- 時間経過で回復 or Super 加入で無制限
- **集中度を維持** + アップセル機構

## 6. 筋肉塾への適用ポイント ⭐⭐⭐

筋肉塾の **「実施完工率を最大化」** という主軸KPIに対して、Duolingo は世界最強の参考。

### ✅ 採用すべき（強推奨）
1. **Streak（連続記録）の導入**
   - 「連続◯日 視聴」「連続◯日 何かしらアクション」
   - 全画面に小さく表示し続ける
   - 損失回避心理で離脱阻止
2. **「ハードル下げる柔軟性 × Streak の絶対性」**
   - 1日に「動画3秒視聴 or できた！1個 or コメント1件」で Streak 維持
   - 重要なのは「途切れさせない」こと
3. **Streak Freeze（救済アイテム）**
   - 月◯回まで「お休み権」を使える
   - 月次振り返りで満タンチャージ
4. **XP / レベル → Skool との統合**
   - 視聴・できた！・コメント・試験合格 全部に XP
   - レベル達成で「次の章解放」や「特典コンテンツ解放」
5. **柔らかい通知設計**
   - 「のり氏が待ってます…」的なキャラ性ある通知
   - ユーモア・寛容なトーン
6. **Built-in Breaks（休憩日設計）**
   - 「今日は1日休みましょう」と通知する日があってもいい
   - 燃え尽き防止
7. **古い章は復習促進（黄色表示等）**
   - 過去のレッスンに戻りたくなる視覚的フック

### ⚠️ 要調整
- **League（リーグ）**: 受講生数によっては成立しない
  - 月13人〜22人規模なら「同期内の活動量グラフ」程度に抑える
- **Hearts（体力制）**: 学習を罰則的にする
  - 筋肉塾には合わない、不採用
- **キャラ性**: のり氏のキャラ自体が代替になる

### ❌ 真似しないほうがいい
- **強迫的な Streak 圧力**: ボディメイクは「短期で結果」ではなく「健康的継続」
  - Streak 失っても自責にさせない設計が必須
- **間違いに罰則（Hearts 制）**: 試験で間違えても挑戦できる優しさを保つ

## 7. アンチパターン

| 由来 | やらないこと |
|---|---|
| Duolingo | Streak を絶対視 → 失った時の自責で離脱増 |
| Duolingo | Hearts のような体力制限 → 学習を「課金で解消する苦行」化 |
| Duolingo | リーグでの過度な競争 → 筋肉塾は「自分との対話」テーマ |
| Duolingo | 通知を多用しすぎる → 鬱陶しさで通知 OFF され、結果離脱 |
