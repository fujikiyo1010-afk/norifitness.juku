画面の役割

  • コース内の各章の詳細画面 ・ 章のレッスン一覧を表示
  • 動線: ホーム → コース一覧 → コース詳細 → 章詳細 → レッスン詳細 (3 階層)
  • 受講生が学習を再開する時の主要動線 (続きからボタン経由)

構成 (上から)

  1. ヘッダー: ← + 章名 + サブ「コース X ・ 第 Y 章」
  2. ブレッドクラム: ホーム / コース / コース名 / 章名 (current)
  3. ヒーロー (ベージュグラデ): CHAPTER N + 章タイトル + 章説明 + 進捗バー
  4. 続きから CTA (ティール緑グラデ): 最後に見たレッスン即ジャンプ + 「▶ 続きから」 + 視聴時間
  5. レッスンリスト (全 N 件):
    • 完了: ✓ アイコン (ティール緑) + done-text スタイル + 「振り返り済」黄バッジ (記入済の場合)
    • 視聴中: ▶ アイコン (緑枠) + 「視聴中」ラベル + 「N 分 (M 分視聴済)」 + ボーダーハイライト
    • 未: 空円 + 通常表示
    • 各カード: LESSON N / タイトル / 動画長 / 振り返り済 / 説明 / タグ
  6. 章末モーダル動線 (黄バナー): 全レッスン完了で章末メッセージ表示の説明
  7. 下部タブ: コース active

含まれる要素 (確定 2026-06-10)

きよむさん指示で全要素を入れて目視で判断する方針:
・ 動画長さ表示 (時計アイコン + N 分)
・ 続きから CTA (大きいティール緑ボタン、 即ジャンプ)
・ 振り返り済バッジ (黄バッジ + ✓ アイコン)
・ 章末モーダル動線 (黄バナー、 完了時の動線案内)
・ 章説明文 (実装通り、 ヒーローに表示)

状態バリエーション (実装で動的)

  • 未着手: 続きから CTA → 「Lesson 1 から始めましょう」、 全レッスン空円
  • 視聴中 (このモック): 続きから CTA → 視聴中のレッスン、 完了 2、 視聴中 1、 未 3
  • 全完了: 続きから CTA → 「もう一度学ぶ」、 章末モーダル表示

未確定 / 議論ポイント

  • 「振り返り済」バッジの位置: meta 行 (現状) or 別行?
  • 動画長さ「N 分 (M 分視聴済)」の表示: 詳細すぎ? シンプル化案あり (「視聴中 50%」プログレスバー等)
  • タグ表示: 全レッスン に出すか、 重要レッスンのみか
  • 章末モーダル動線: 「全完了時に出る」の説明バナー、 不要なら削除
  • レッスンカードのホバー効果: モバイルでは無効化
