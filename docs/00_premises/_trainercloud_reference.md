## 📱 trainercloud（MyTrainer）の概要

- **正式アプリ名**: MyTrainer
- **ブランド表示**: BodyChanger（のりfitness）
- **ドメイン**: `app.trainer-cloud.com`
- **メール送信元**: `@trainer-cloud.com`
- **形態**: OEM 型サービス（既存 SaaS を契約してカスタマイズ運用）
- **対応プラットフォーム**: iOS（App Store）/ Android（Google Play）

### 業界・対象
- パーソナルトレーニング業界向け
- 「トレーニー」「トレーナー」「ジム」という用語使用
- 1人のトレーニー（受講生）に対し、専属パーソナルトレーナー（指導者）が紐づく構造

## 🛠 アプリ機能一覧（スクショから判明）

### 受講生側の機能

| カテゴリ | 機能 |
|---|---|
| **ホーム** | 本日のサマリー（体重・体脂肪率・カロリー・PFC・コンプリート表示）、3つの記録カード（体重/食事/トレーニング）、統計グラフ（体重・体脂肪率推移）|
| **体重・ライフログ記録** | 体重・体脂肪率・ウエスト・体型写真・気分（5段階）・お通じ・睡眠時間・飲酒・日記 |
| **食事記録** | 食事タイプ（朝食/昼食/夕食/間食/まとめ）・写真・カロリー・PFC バランス・メモ |
| **トレーニング記録** | 種目選択・セット数・距離・時間・完了チェック・履歴から追加・種目を追加 |
| **エクササイズ DB** | 部位別検索（腕/脚/ふくらはぎ/背中/胸/臀部/肩/腹筋/有酸素/その他）、自重系豊富（プッシュアップ・スクワット・ヒップリフト・クランチ・レッグレイズ・プランク等）|
| **カレンダー** | 日別の全データ統合ビュー、週間表示 |
| **メッセージ** | トレーナーとの双方向チャット |
| **目標設定** | 身長・目標体重・目標体脂肪率（オンボーディング時）|
| **プロフィール編集** | 名前・メアド・性別・生年月日・身長・自己紹介・アバター画像 |
| **通知設定** | プッシュ通知 ON/OFF、メッセージ・フィードバック・記録リマインダー（時刻設定可）|
| **パスワード変更** | 現在のパスワード + 新しいパスワード（大・小・数字含む6文字以上）|
| **設定** | よくある質問・利用規約・プライバシーポリシー・アプリに関するお問い合わせ・アプリについて |

### 管理者側の機能
- 招待メール送信（有効期限 7日間）
- 招待リンク発行（`app.trainer-cloud.com/invite/prd?token=...&email=...`）
- 受講生情報の閲覧・編集
- メッセージ送信
- フィードバック送信
- データ閲覧（食事・運動・体重等）

## 📨 招待フロー（trainercloud 側）

```
1. 管理者が受講生を招待
   ↓
2. 受講生にメール送信（BODY CHANGER / トレーニングのご招待）
   - 「のりfitness さんから、トレーニング管理アプリへの招待が届いています」
   - 「招待を開く」ボタン
   - 有効期限: 7日間
   - 本文に直リンク併記（ボタン非対応時用）
   ↓
3. 受講生がリンククリック → Web画面
   - 「My Trainer に参加」
   - 「fujikiyo1010@gmail.com 宛の招待です」
   - 「スマートフォンからアプリを開いて、アカウント連携を完了してください」
   ↓
4. アプリインストール誘導
   - iOS用アプリ（App Store）
   - Android用アプリ（Google Play）
   ↓
5. アプリ起動 → アカウント連携 → ログイン
   ↓
6. オンボーディング
   - 身長入力
   - 目標体重設定（スキップ可能）
   - 目標体脂肪率設定（スキップ可能）
   - 目標設定の確認 → 保存
   ↓
7. 利用開始
```

## ❓ FAQ 全文（参考）

> MyTrainer のご利用に関して、よく寄せられるご質問とその回答

### はじめに

**Q. MyTrainer とはどのようなアプリですか？**
A. MyTrainer は、パーソナルトレーニングの効果を最大化するための専用アプリです。
日々の食事記録や体重・体脂肪率の記録、トレーニング内容の確認、そして担当トレーナー
とのチャットコミュニケーションをひとつのアプリで行うことができます。

**Q. アプリの利用に料金はかかりますか？**
A. トレーニー（ユーザー）の皆様にとってアプリの利用自体は無料です。ただし、通われて
いるパーソナルトレーニングジムや契約されているトレーナーとのご契約に関する費用は別途
発生いたします。

**Q. 招待メールが届かないのですが？**
A. 迷惑メールフォルダやプロモーションフォルダに振り分けられていないかご確認ください。
「@trainer-cloud.com」からのメールを受信できるように設定をお願いいたします。
それでも届かない場合は、担当のトレーナーまたはジムにご連絡いただき、メールアドレスに
誤りがないかご確認ください。

### 記録・機能について

**Q. 食事の記録はどのように行えばよいですか？**
A. 「食事記録」メニューから、朝食・昼食・夕食・間食ごとに記録を追加できます。
カロリーや PFC バランスなどの数値入力も可能です。担当トレーナーがあなたの記録を確認し、
アドバイスを行います。

**Q. 体重や体脂肪率のグラフはどこで見られますか？**
A. （回答未記載 / 推測: ホーム画面の「統計グラフ」または「カレンダー」画面）

**Q. 前の日の記録を修正することはできますか？**
A. （回答未記載 / 推測: カレンダー画面から過去日付を選択 → 編集可能）

### メッセージ・通知

**Q. トレーナーからの返信はいつ来ますか？**
A. トレーナーの就業時間や方針により異なります。通常、トレーニング外の時間に確認・返信
させていただきますので、担当トレーナーにおおよその返信タイミングを事前にご確認ください。

**Q. アプリの通知が来ないのですが？**
A. スマートフォンの「設定」アプリから、myTrainer の通知が許可されているかご確認ください。
Web ブラウザ版をご利用の場合は、ブラウザの通知設定もあわせてご確認ください。

### アカウント・その他

**Q. パスワードを忘れてしまいました。**
A. ログイン画面の「パスワードを忘れた場合」から、ご登録のメールアドレスを入力し、パスワード
再設定のメールを受け取ってください。メール内のリンクから新しいパスワードを設定できます。

**Q. 担当トレーナーが変更になった場合はどうなりますか？**
A. ジム側での設定により、新しい担当トレーナーに自動的にデータが引き継がれます。
ユーザー様側で特別な操作をしていただく必要はありません。

**Q. 退会（アカウント削除）したいのですが。**
A. 現在通われているジム・パーソナルトレーナーとの契約を解除された場合、トレーナー側の
管理画面からアカウントの停止処理が行われます。アプリからのデータ完全削除をご希望の場合は、
お問い合わせフォームより直接ご連絡ください。

## 🎯 新サイト起草時の参考ポイント

### プライバシーポリシー（新サイト用）
trainercloud の構造を踏襲しつつ、**学習プラットフォーム独自の項目**を追加:

| trainercloud | 新サイト（変換） |
|---|---|
| 身長、体重、体脂肪率等の身体情報 | → **学習進捗データ・視聴履歴・試験結果** |
| 食事内容、トレーニング履歴 | → **目標管理シート・コメント投稿** |
| 担当トレーナーとのメッセージ通信履歴 | → **LINE 連携経由のコミュニケーション履歴** |
| 第三者提供: パーソナルトレーナー・ジム | → **第三者提供: 食事専属スタッフ・運営チーム** |

### 利用規約（新サイト用）
trainercloud の構造そのまま流用可能、独自項目を追加:
- **コメント機能の規約**（公開コメント、不適切投稿の削除）
- **学習データの取扱**（永久対応の範囲、卒業後のアクセス権）
- **10日間全額返金保証**の規定（学習目的達成不可時の対応）
- **既存 saipon の利用規約**（規約違反は強制退会等）も統合
