# 11 深掘り: SHElikes（シーライクス）

**URL**: https://shelikes.jp/
**料金**: 入会金162,800円 + 月額10,780円〜（年間約35〜40万円）
**販売形態**: 入会金型サブスク（補助金で最大70%還元）
**筋肉塾との関係**: **コーチング付き学習スクール**、構造が最も類似（入会金 + 学習空間）

---

## 1. サービス概要・ターゲット

| 項目 | 内容 |
|---|---|
| 概要 | 女性向けキャリアスクール。50種類以上のWebデザイン・マーケ・ライティング等のコース |
| 主な対象 | キャリアアップ・副業希望の20-40代女性 |
| 強み | 「動画学習 + 月次コーチング + コミュニティ」の三位一体 |

---

## 2. 会員サイトの画面構成（マイページ）

### マイページの主な機能
1. **コース動画視聴**（学習の中心）
2. **コーチング予約**（月1回、TAから日時とコーチを選んで予約）
3. **もくもく会予約**（複数人で同時作業＋TA質問対応）
4. **イベント予約**（特別イベント、入学式、グループ学習プログラム）
5. **コミュニティ**（プラン共通でアクセス可）

→ つまり **「動画 + ライブ予約 + コミュニティ」の3ハブ**

### 学習スタイル
- 動画学習が中心
- **学習スケジュールは自己管理**（強制的なドリップ配信は薄め）
- 課題提出制度あり（コースによる）

---

## 3. 学習継続を促す仕組み ⭐ 筋肉塾の最大の参考

### コーチング月1回（TA面談）
- TA（ティーチング・アシスタント）= コーチ
- マイページから日時・コーチ選択 → 予約
- 内容: 進捗確認、今後のアクションプラン相談
- オンライン（Zoom）or 拠点（青山・銀座・名古屋・梅田）
- **「進捗確認 + 今後のプラン」= サクセスラーニング型の動力源**

### もくもく会
- 複数人で同時作業（Zoom等）
- TA に質問可能
- **コミュニティ密度を上げる装置**

### 課題提出 + フィードバック
- 一部コースでは課題ありき
- 提出 → TA確認 → コメント

### コミュニティ（Slack風 or 専用）
- プラン共通の交流空間
- 受講生同士の質問・励まし合い

---

## 4. 試験・テスト機能

- 明示的な「合否+正答率」型の試験は**主軸ではない**
- 課題提出 + TA フィードバックが代替
- コース完了マーク（バッジ的）はあり（推測）

---

## 5. コミュニティ機能

- プラン共通の「コミュニティ」（108件の記事で説明されるほど構造化されている模様）
- イベント（入学式、グループ学習、特別イベント）が定期開催
- 「同期感」を作るオフライン拠点（東京・名古屋・大阪）

## 7. 筋肉塾への適用ポイント ⭐⭐⭐

SHElikes は構造が筋肉塾と**最も類似**。以下を直接参考に。

### ✅ 採用すべき
1. **マイページの3ハブ構成**
   - 動画コース / コーチング予約 / コミュニティ
   - 筋肉塾: コンテンツ / 目標シート / コメント（仲間との学び）の3軸に翻訳可能
2. **月次の「進捗確認 + アクションプラン」**
   - 筋肉塾は1on1コーチングないが、**目標管理シートの月次レビュー（Should Have）** が代替
   - 管理者（社長/きよむさん）からのコメントが「TA面談」相当
3. **オンライン or オフラインの選択肢**
   - 筋肉塾もオフライン合宿の余地あり（将来）
4. **入会金 + 月額の二層構造**
   - 筋肉塾は買い切り型なので適用しないが、「Should Have で永久会員化」の参考に
5. **課題提出 + フィードバック**
   - 筋肉塾の「できた！ボタン」のさらに上位機能として、「課題提出 → コメント返信」も将来あり得る

### ⚠️ 要調整
- **コーチング個別予約UI**: 筋肉塾は LINE経由なのでマイページ内予約は不要
- **複数コース選び**: 筋肉塾は5大カテゴリで固定、SHElikes 50+ コースのような豊富さは不要

### ❌ 真似しないほうがいい
- **コーチング月1回が必須化**: 筋肉塾はサポート期間180日後は学習者自走、強制面談は合わない
- **自己管理スケジュール（強制ドリップ薄め）**: 筋肉塾は段階公開で「次が出る期待感」を維持する戦略

## 8. アンチパターン

| 由来 | やらないこと |
|---|---|
| SHElikes | コース数を増やして選択肢を膨らます → 完工率低下リスク |
| SHElikes | 「自己管理」を建前にして放置 → 離脱率増 |
| SHElikes | コーチング予約を自分で取らせる手間 → 筋肉塾は LINE 飛ばしのシンプル設計で良い |
