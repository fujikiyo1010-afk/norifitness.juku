🗺 構成 (上から順)

  • ① ヘッダー帯: タイトル + 提出日 + 入会日数 + 記入状況
  • ② KPI サマリーバー: 4 カード (現状体重 / 目標体重 ★ / 期限 / 目標 kcal)
  • ③ 総評バー: のりfitness の全体総評 (薄黄背景)
  • ④ 本体カルテ表: 5 セクション、左列ラベル + 右列値 + 各セクション末尾に添削行
  • ⑤ フッター: 編集履歴/変化を見る/過去の添削履歴 リンク + 編集するボタン

🎨 デザイン特徴

  • 契約書/カルテ的 なドキュメント感 (フォーム入力 UI ではなく、結果一覧として読ませる)
  • セクション帯 = 黒背景 + 白文字 + 「01 02...」(2桁) + 文字小さめ + letter-spacing で業務感
  • 左列ラベル = 薄灰背景 + グレー文字、右列値 = 白背景 + 黒文字
  • 添削行 = 黄背景セル (v3 と統一)
  • 目標体重カードのみ v5 強調 (数値の下にティール緑下線、背景・文字色は他と同じ)
  • 編集するボタン = ティール緑 + 角丸 3px (シャープ)

🔗 動線 (実装時)

  • ホーム「🎯 目標管理シート」横長ブロック → このページ (閲覧モード) がデフォルト
  • 「編集する」ボタン → 編集モード (v3 スマホ版) へ遷移
  • 「編集履歴」→ goal_sheet_revisions の履歴画面
  • 「変化を見る」→ マイ学習ログの 5 枚目カード
  • 「過去の添削履歴」→ goal_sheet_comments の履歴一覧
  • URL: /goal-sheet (閲覧) / /goal-sheet/edit (編集)

🖨 印刷対応

  • Cmd+P (Mac) / Ctrl+P (Win) で印刷ダイアログ
  • A4 縦 1-2 枚に収まるよう @media print で調整済
  • 印刷時は注釈・ページヘッダー・編集するボタンは非表示

📱 スマホ対応

  • 720px 以下で KPI バーは 2×2 グリッドに
  • ヘッダーは縦並びに
  • カルテ表のラベル列は狭く、フォントサイズ縮小
  • フッターは縦並びで編集ボタンは全幅
