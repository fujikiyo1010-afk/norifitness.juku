🎬 紙吹雪アニメーション仕様

  • CSS のみで実装 (JS 不要、軽量)
  • 20 粒、それぞれ animation-duration と animation-delay がランダム
  • 3.2s〜4.5s で落下、ランダム遅延 0〜2.0s
  • 落下 + 720度回転 + 透明度 0.9 → 0.4
  • infinite ループ = 継続的に降り続ける (ユーザーが画面を閉じるまで)
  • ※ 実装時は「最初の 5 秒だけ」など制御可能

📝 メッセージのトーン

  • 「完了!」は許容 (5 回しかない節目なので "!" 使ってもOK)
  • 「学び終えました」 = 事実を肯定

  • サマリ数値はポジティブな指標のみ (学習日数 / 視聴時間 / 完了レッスン)

📊 サマリ数値 (合意済仕様)

  • 視聴時間: コース内全レッスンの動画視聴時間 (例: 4.5 時間)
  • 完了レッスン: 「できた!」を押したレッスン数 (例: 32 個)
