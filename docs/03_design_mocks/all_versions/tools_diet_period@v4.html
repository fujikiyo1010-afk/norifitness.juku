🗺 画面の役割

  • ツール一覧 → 「③ 減量期間逆算」タップ → このページ
  • 目標体重 + ペース + 開始日 から、到達予定日と週次タイムラインを算出
  • 反映 → 目標シート ② 目標選定「目標体重 / 目標期間 / 目標達成日」に転記

📐 構成 (上から)

  • ヒーロー: ツール番号 + タイトル + 前回計算日
  • 入力カード: 現在体重 / 目標体重 / 1週間のペース / 開始日
  • 計算ボタン: インディゴ + 3px
  • アドバイス: ペース別動的メッセージ (≤0.5 / ≤1.0 / >1.0)
  • 結果カード: 必要な減量 + 期間 + 到達日 + 進捗バー
  • タイムライン: 週ごとカード + 節目ラベル (1ヶ月/3ヶ月/...)
    • 節目週 = インディゴ系背景
    • 目標到達週 = 黄色系背景 (達成感)
  • PDF/印刷ボタン: 特典 24 の機能を継承
  • 反映ボタン: ティール緑 + 3px

🧮 計算ロジック

  • 必要な減量 = 現在体重 − 目標体重
  • 期間 (週) = 減量 / ペース
  • 期間 (日) = 週 × 7
  • 目標到達日 = 開始日 + 日数

💡 ペース別アドバイス (特典 24 継承)

  • ≤ 0.5 kg/週: 「無理の少ない、続けやすいペースです」
  • ≤ 1.0 kg/週: 「ややしっかりめ、体調を見ながら」
  • > 1.0 kg/週: 「かなり速い、計画見直しを」

📅 節目ラベル

  • 5 週目 → 1 ヶ月の目安
  • 13 週目 → 3 ヶ月の目安
  • 21 週目 → 5 ヶ月の目安
  • 29 週目 → 7 ヶ月の目安
  • 37 週目 → 9 ヶ月の目安
  • 最終週 → 🎯 目標到達の目安 (黄色背景)

📤 反映内容

  • 目標シート ② 目標選定「目標体重」← 入力値
  • 目標シート ② 目標選定「目標期間」← 週数
  • 目標シート ② 目標選定「目標達成日」← 到達日
