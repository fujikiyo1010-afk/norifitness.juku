# きよむさん用 TODO — 社長確認の前に自分で調べる項目

**作成日**: 2026-05-18
**目的**: 社長への質問数を減らすため、**きよむさん自身がエキスパ・運用システム経由で調べられる項目**をリストアップしました。
**進め方**: 1項目ずつ調べて、分かった内容を本ファイル下部の「調査結果メモ」に書き込んでください。私がそれを見て次に進めます。

## 📌 調査項目

### 1. 動画コンテンツのホスト場所
**何を調べる**: 筋肉塾サイト(`saipon.jp/h/dyz555`)に埋め込まれている**動画の実体がどこにあるか**

**調べ方候補**:
- エキスパ管理画面 → サイポン → オンラインサロン → 任意のレッスンを「編集」で開く
- 動画埋め込み箇所の URL を確認（YouTube / Vimeo / 自社サーバー / その他）
- 動画プレイヤーの右クリック → 「ビデオの URL をコピー」で確認できる場合あり

**期待される回答例**:
- 「Vimeo です。アカウント: xxx@xxx.com」
- 「YouTube の非公開動画です」
- 「サイポンに直接アップロードしています」
- 「Wistia / Mux / その他: ＿＿＿」

**なぜ必要**: 新サイトでの動画配信方式の選定に直結。既存動画を移行するコスト見積もりに必要。

---

### 2. 現在の決済プロバイダ
**何を調べる**: 受講生がクレカ決済する時、**どの会社のシステムを使っている**か

**調べ方候補**:
- 銀行口座への入金履歴を確認（入金元の名義に決済代行会社名が出る）
  - 例: 「ストライプジヤパン」「ユニヴァペイキャスト」「ペイジエンドット」「エキスパ決済」等
- 過去の受講生への請求書・領収書を確認
- 会計ソフトの取引履歴を確認
- 社長または経理担当者が契約書を持っている可能性

**期待される回答例**:
- 「Stripe」
- 「UnivaPay」
- 「Pay.jp」
- 「エキスパの決済機能」
- 「PayPal」
- 「複数併用: ＿＿＿ + ＿＿＿」

**なぜ必要**: 新サイトの決済選定 + 既存決済データの引き継ぎ可否判断に直結。

---

### 3. 受講生マスタの実体的な所在
**何を調べる**: 受講生の氏名・連絡先・入会日・プラン等の**マスタデータがどこにあるか**

**背景**: エキスパのサロン管理（β版）の会員一覧は「招待会員・期限切れ」だらけで実質未使用と判明。**実際の受講生情報は別の場所にある**はず。

**調べ方候補**:
- きよむさん普段の受講生管理業務で開いているシステム/ファイルを思い出す
- スプレッドシート / Notion / Airtable / その他 CRM ツール?
- LINE 公式アカウントの友だちリスト?
- 決済プロバイダ側の顧客リスト?
- 社長専用の Excel ファイル?

**期待される回答例**:
- 「Google Sheets で管理しています、URL: xxx」
- 「Notion のデータベースです」
- 「LINE 公式アカウントの友だちタグで管理」
- 「決済プロバイダ側のみ。それ以外には台帳なし」
- 「複数の場所に散らばっている」

**なぜ必要**: Supabase のテーブル設計 + 既存会員移行のソースデータ特定に直結。

---

### 4. リバウンド保証の正確な規定
**何を調べる**: 「リバウンド保証」の**正確な条件**

**調べ方候補**:
- 利用規約のリバウンド保証セクションを確認
- 販売LP の保証文言を確認
- セミナーの価格スライドに記載があるはず
- 過去に返金処理した記録があれば、そこから条件を逆算

**確認したい項目**:
- 何日以内に申請可能？
- 何の条件を満たせば返金対象？
- 全額返金 / 部分返金?
- 申請方法は?（LINE? メール? フォーム?）

**期待される回答例**:
```
- 申請期限: 入会から〇日以内
- 適用条件: ○○を満たした上で結果が出なかった場合
- 返金額: 全額 / 50% / etc
- 申請方法: ○○
```

**なぜ必要**: 新サイトの DB 設計（リバウンド保証管理機能）に直結。

---

### 5. 「チェンジャー」(`spn-apr.jp/h/norifitness`) と筋肉塾の関係
**何を調べる**: 限定特典ページに「**オンラインジムチェンジャー無料招待**」というリンクがあり、別サービスのよう。これと筋肉塾の関係。

**背景**: 受講生視点TOP画面のフッターにも「オンラインジム チェンジャー」表記あり。`saipon.jp/h/bps543`（株式会社ボディーチェンジャー）と関係があるかも。

**確認したい項目**:
- 「チェンジャー」とは何のサービス?
- 筋肉塾受講生は自動的にチェンジャーにも入れる?
- チェンジャーの会員と筋肉塾の会員は別々?
- これも新サイトに統合する必要がある?

**調べ方候補**:
- spn-apr.jp/h/norifitness を実際に開いてみる
- きよむさんが運営に関わっているなら、内部資料を確認
- 社長が販売しているサービスマップを思い出す

---

### 6. norifitness.com の管理主体・移行方針
**何を調べる**: 限定特典ページに `norifitness.com/ietre/` や `norifitness.com/11556-2/` 等のリンクが多数。**このサイトは誰が管理しているか**

**確認したい項目**:
- 誰が管理? (社長? きよむさん? 外注?)
- 何で作られている? (WordPress? 静的サイト? その他?)
- ホスティング先は?
- 編集権限は誰が持っている?

**調べ方候補**:
- きよむさんが普段触っているなら知っている
- 社長 or きよむさんで管理画面にログインしてみる
- ホスティング契約書 / WordPress 管理画面の URL を探す

**なぜ必要**: 新サイトとの統合 or 別運用の判断材料。**私の推奨は "外部リンクとして残す"**（統合しない）が、社長の意向次第。

---

### 7. trainercloud のリリース時期・開発進捗
**何を調べる**: trainercloud の**現在の開発状況**と**リリース予定**

**背景**: noriAI_business.md には「未リリース、自社利用のみ、まず筋肉塾生徒に配布予定」とあるが、最新の状況は不明。

**確認したい項目**:
- 現在の開発進捗 (何%完了?)
- 動いている機能 / 動いていない機能
- リリース予定日
- 新サイトとの SSO 連携を、いつ頃から組めるか

**調べ方候補**:
- 社長 or 開発担当者に確認
- trainercloud のリポジトリ / 管理画面があれば確認

**なぜ必要**: 新サイトの段階リリース計画に直結。フェーズ3（SSO 連携）の時期を決める材料。
