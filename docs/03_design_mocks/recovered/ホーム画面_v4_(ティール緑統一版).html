ホーム画面 v4 (最新合意反映版)

  • 旧 home_v3.html (明るい緑 #4ECD5D) を ティール緑 #00897b に統一
  • Phase 2-7 メモリの全合意事項を反映
  • 受講生 UI の中核、毎日見る画面

構成 (上から)

  • ヘッダー: 筋肉塾 ロゴ + ✉ お知らせ (赤ドット) + 👤 プロフィール
  • ヒーロー: 「こんにちは、◯◯ さん」+ 入会日 + 経過日数 (E案グラデ) ・ 氏名は会員情報から自動反映 (申請フォーム → 会員情報登録 → 表示)
  • 黄バナー: 目標管理シート未記入時のみ表示 (フラット文言、閉じれない、タップで遷移)
  • 続きから学ぶ CTA: 緑グラデ + 章/レッスン名 + ▶ アイコン (一番目立つ)
  • 4 機能ブロック: コース / 学習 / LINE / ツール (角丸 4px シャープ、線画黒一色)
  • 横長: 目標管理シート (角丸 14px、のり社長返信時は 📩 バッジ)
  • 全体進捗: 35% + 76 / 217、進捗バー
  • 数値 3 枠: 完了レッスン / 視聴時間 / 入会日数
  • 下部 5 タブ: ホーム (active) / コース / 記録 / 検索 / 設定

デザイン適用 (Phase 2-7 メモリ)

  • メイン色: ティール緑 #00897b (旧 #4ECD5D から変更)
  • 機能ブロック: 角丸 4px (シャープ、Nike エッセンス)
  • その他カード: 角丸 12-14px (CTA, 横長, 進捗, 数値カード)
  • 影: A 版基準 (CTA のみ強め、他は薄め)
  • アイコン: 線画黒一色 (Lucide / Heroicons outline)
  • フォント: Noto Sans JP + Inter (数値・日付)

ヘッダーアイコン挙動

  • ✉ お知らせ → 赤ドット = 未読あり、タップで通知一覧
  • 👤 プロフィール → タップで profile_screen へ遷移

黄バナーの仕様 (合意済)

表示条件: 目標管理シートが未記入の時のみ
文言: 「目標管理シートが未記入です。タップして設定しましょう」
挙動: 閉じれない (毎ログイン視覚的催促)、タップで目標シート編集画面へ
記入完了後: バナー自動非表示
色: 添削色 (黄) + ダークゴールド = 注意喚起だが柔らかい

横長ブロックの動的バッジ

  • のり社長から添削が届いている時に 📩 1 のバッジ表示
  • 未読数を数字で表示
  • タップで目標シート閲覧モード → 添削箇所にハイライト

数値 3 枠の指標

  • 完了レッスン (76 / 全 217 のうち)
  • 視聴時間 (累計、ポジティブ指標)
  • 入会から日数 (経過日数、自然な指標)
  • ※ 「サボった日数」「未完了レッスン数」など否定的指標は出さない (責めない方針)

下部タブ 5 個 (確定)

  • 🏠 ホーム (active)
  • 📚 コース
  • 📝 記録 (体組成記録 + 月次添削)
  • 🔍 検索
  • ⚙ 設定

状態バリエーション (実装時)

  • 状態 1: 入会直後 (黄バナー表示、「続きから学ぶ」は「最初のレッスンへ」に変化)
  • 状態 2: 目標シート記入済 (黄バナー非表示)
  • 状態 3: 添削届いた (横長に 📩 バッジ + ヘッダー ✉ に赤ドット)
  • 状態 4: 通常 (現在のモック表示)

Phase 2 候補

  • キャラ画像をホームに入れた場合の見た目 (「こんにちは」の右に小キャラ)
  • 下部タブのアイコン微調整 (線画統一の徹底)
  • 進捗バー / 数値カードのインタラクション (タップで詳細画面)
