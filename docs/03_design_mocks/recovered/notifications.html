画面の役割

  • ホーム画面のヘッダー右上 ✉ アイコンから開く
  • 受講生に届く通知 (動画返信 / メニュー配布 / 月次受付開始 / 目標シート添削 / メンテナンス等) を一覧で確認

構成 (上から)

  • ヘッダー: ← + 「お知らせ」(中央寄せ)
  • カード (白背景、border-radius 14px):
    • 左上に薄ピンクタグ (種別) 例: メンテナンス / 月次添削 / 筋トレメニュー / 目標シート / お知らせ
    • タイトル (太字 14px)
    • 日付 (グレー、Inter フォント)
    • 右に chevron `>` (タップ可示唆)

動線

  • 各通知をタップ → 該当画面 (動画返信なら月詳細、メニューなら筋トレメニュー等)
