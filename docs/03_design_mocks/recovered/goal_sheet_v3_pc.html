🗺 PC 一枚ものビュー構成

  • 上部固定ヘッダー: 戻る / タイトル / 進捗バー / 下書き保存 / 保存する
  • 3 カラム配置: 左 (①②) / 中央 (③ 栄養設計) / 右 (④⑤ + 全体総評 + ボタン)
  • ③ 栄養設計は情報量が多いので、中央カラムを少し広め (1fr 1.2fr 1fr)
  • 下部タブは PC ではサイドナビ前提のため非表示

🖨 印刷対応

  • Ctrl+P (Mac は Cmd+P) で印刷ダイアログ
  • A4 横サイズ 1 枚に収まるよう @media print で調整済み
  • 注釈・ページヘッダーは印刷時に非表示

🎨 配色 (スマホ版と同じ)

  • Nike エッセンス完全準拠 (明るい緑 + グレー罫線 + 黒文字)
  • 添削系 = 薄黄色 (黄マーカー同色)
  • 目標カードのみ深緑→明るい緑グラデ
