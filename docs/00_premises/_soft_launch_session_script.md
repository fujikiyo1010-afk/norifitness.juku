## 🎬 進行台本 ・ 4 Phase / 90 分

### Phase 0 ・ 集合 + 挨拶 (= 10 分)

#### きよむさんの台本

> 「今日は新筋肉塾アプリの β テストです。
> 
> 完璧じゃないので、 違和感あったらすぐ言ってください。 むしろ違和感を出すのが今日の目的です。
> 
> LINE グループ作りました。 何でも投げてください。 『ここ分かりにくい』 『ここ壊れてる』 『ここいい感じ』 全部歓迎です。
> 
> 90 分くらいで、 入会 → 機能体験 → 管理画面 まで一通り触ってもらいます。 まずは iPhone + PC、 ご準備ください」

#### きよむさんが やること
1. LINE グループ作成 → 4 人参加
2. このマニュアルの GitHub URL を LINE で共有 (= [_soft_launch_guide.md](https://github.com/fujikiyo1010-afk/norifitness.juku/blob/main/docs/00_premises/_soft_launch_guide.md))
3. のり氏 + 社員 2 名のメアドを確認 (= 招待発行用)

### Phase 1 ・ 招待発行 (= 10 分)

#### きよむさんが やること

1. Mac で [https://juku.norifitness.com/admin/invitations](https://juku.norifitness.com/admin/invitations) 開く
2. **画面共有 or 大画面に映す** ← 全員に「これが管理画面の招待ページです」 と見せる
3. 「**新規招待**」 セクションで 3 人分順次入力:
   - のり氏: 氏名 + メアド入力 → 送信
   - 社員 A: 氏名 + メアド入力 → 送信
   - 社員 B: 氏名 + メアド入力 → 送信
4. 各人「届きました?」 確認 → Gmail で受信確認してもらう
5. **きよむさん自分用も発行** (= 例: `fujikiyo1010+kiyomu-test@gmail.com`)

#### きよむさんの台本

> 「今招待メールを送りました。 Gmail を見てください。
> 
> 件名は『【筋肉塾】受講開始のご案内』 です。 30 秒以内に届くはずです。
> 
> 届いたら 開かないで待ってください。 次の説明をしてから 同時に進めます」

---

### Phase 2 ・ 受講生 動線 (= 40 分 ・ 各自 iPhone で)

#### きよむさんの台本 (= 開始時)

> 「ここから 40 分、 各自 iPhone で進めてください。 
> 
> 私は LINE 見ています、 詰まったら投げてください。 
> 
> 順番は:
> 1. メール開く → リンクタップ
> 2. パスワード設定
> 3. オンボーディング (= 8 ステップ)
> 4. ホーム画面追加 → アプリ再起動
> 5. カルテ提出
> 6. 目標シート記入
> 7. 体組成 1 回記録
> 
> です。 詳細は LINE のマニュアル URL に書いてあります。 では どうぞ」

#### Phase 2 ・ きよむさん 並行で やること
- LINE 観察 (= 即フォロー)
- 自分の iPhone でも同じ動線 体験 (= 自分が初めて触る気持ちで)
- 詰まった人に「ここ押してみてください」 案内
- バグ報告は **その場で Claude に共有** (= 重要度高い場合)

#### Phase 2 ・ よくある詰まりどころ + 対処

| 詰まりどころ | きよむさんの対処 |
|---|---|
| 「メール 来てない」 | Gmail 検索「筋肉塾」 / 迷惑メールフォルダ確認 |
| 「Chrome で開いた、 どうする?」 | 「Safari で開いてください」 案内画面の手順に従って |
| 「PWA 化できない」 | iPhone Safari の共有ボタン (画面下中央 / 四角に上矢印) を確認 |
| 「カルテ提出後何が起きる?」 | 「管理画面で『メニュー作成中』 になります、 後でメニュー配布します」 |
| 「目標シートの項目多すぎる」 | 「必須項目だけで OK / 任意は飛ばして」 |

---

### Phase 3 ・ 管理画面 体験 (= 30 分 ・ 各自 PC + きよむさんがガイド)

#### Phase 3-A ・ きよむさん画面共有 で admin 紹介 (= 10 分)

1. [https://juku.norifitness.com/admin](https://juku.norifitness.com/admin) を画面共有
2. **5 枠カード** を解説:
   - 「今月の月次添削 完了率」
   - 「月次添削 未送信」
   - 「重要アラート」 (= 体組成 7 日途絶 / 目標シート乖離)
   - 「リクエスト未対応」
   - 「プロテイン発送未対応」
   - 「新規入会 未処理」
3. [https://juku.norifitness.com/admin/users](https://juku.norifitness.com/admin/users) 受講生一覧 表示
4. **8 カラム** を解説:
   - 受講生 / カルテ / 目標シート / 月次 / メニュー / リクエスト / 最終アクション / 操作
5. 「**ハブを開く →**」 でリハ太郎 (= 自分) のハブ表示 → 各タブ紹介

#### Phase 3-B ・ きよむさん実演 (= メニュー配布) (= 10 分)

1. リハ太郎 (= 自分) のハブ → 「マッチング検索から新規配布 →」 タップ
2. /admin/users/[id]/match 画面 = マッチング候補表示
3. 1 位を採用 → 配布画面
4. のり氏 メモ追加 (= 例: 「腹筋追加しました」)
5. **「配布する」** タップ → 自分の iPhone に Push 通知が届く実演

#### Phase 3-C ・ のり氏 + 社員 体験 (= 10 分)

選択肢 A (= 社員 2 名 admin 一時追加した場合):
- 全員 PC で [https://juku.norifitness.com/admin](https://juku.norifitness.com/admin) ログイン
- 各画面 自由に触る (= /admin/users / /admin/requests / /admin/shipments / /admin/messages 等)

選択肢 B/C (= 社員 2 名 admin なし):
- のり氏 が PC で /admin ログイン → 触る
- 社員 2 名 は きよむさんの画面共有を見るだけ

### Phase 4 ・ 振り返り + 質問 (= 10 分)

#### きよむさんの台本

> 「お疲れさまでした。 一通り体験できましたか?
> 
> 1 人ずつ印象を教えてください。 良かった点 / 違和感 / こうしてほしい点 何でも OK です。
> 
> 後から気付いたことは LINE に投げてください。
> 
> このまま 1-2 週間 継続テストです。 普段使いの感覚で触ってみてください。 リマインド通知 とかも自然に来ます。

> ありがとうございました!」

- 全員の印象を 1 言ずつ聞く (= LINE で書いてもらってもOK)
