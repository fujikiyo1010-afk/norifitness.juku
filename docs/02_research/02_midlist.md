### C-01: WEBMARKS

**URL**: https://webmarks.co.jp/

#### サービス概要
完全オンライン型のWebマーケスクール。SEO・Web広告・生成AIを統合的に学べる実践型講座。現役Webマーケターが指導。

#### ターゲット顧客
- 未経験からWebマーケターを目指す個人
- フリーランス・副業希望者
- 会社員からの転職希望者
- 法人企業の社員研修

#### 料金プラン
| プラン | 価格（税込） | 期間 |
|---|---|---|
| 超実践型講座 | **495,000円** | 6ヶ月 |
| 動画講座 | **385,000円** | 3ヶ月（自主学習） |
| SEO/GA4/広告など個別講座 | 別途設定 | 3〜6ヶ月 |

- 2025年6月からリスキリング補助金適用開始 → 最大100% OFF（598,000円までキャッシュバック）
- 補助金内訳: 経産省 70% + 独自キャッシュバック 30%
- 修了後の転職1年継続でさらに 20% 追加キャッシュバック

#### 主要機能・カリキュラム
- AI×SEO×Web広告 統合講座
- SEOマーケ講座、Web広告講座、GA4スクール、SEOライタースクール
- 200本以上のライトニングトーク動画
- 課題添削

#### 入会導線
```
無料カウンセリング予約
   ↓
現役Webマーケターと面談（オンライン）
   ↓
講座申込
```

#### サポート体制・保証制度
- **返金保証**: 14日以内かつ全課題提出・全授業出席の場合のみ全額返金
- 個別面談「受け放題」
- チャット無制限質問
- 職務経歴書添削
- 懇親会・交流会
- 卒業後も継続的な勉強会
- キャリアアドバイザー伴走

#### UI/UXの特徴
- 受講生専用ポータル・LMS の詳細は公式LP に記載なし
- 公式LP は「補助金訴求」「卒業生実績」「案件獲得率93%」が中心

#### 強み・弱み
- **強み**: 補助金・キャッシュバックを前面、案件獲得率93%・転職実績多数、サポートが手厚い印象
- **弱み**: 受講生体験（LMS・コミュニティ）の中身が外部から見えにくい

#### 軸C 追加項目
- **入会導線**: 無料カウンセリング → 個別面談 → 申込（個別面談必須）
- **解約防止**: 14日返金保証あり（条件厳しめ）
- **サポート**: 個別面談無制限、チャット無制限
- **保証制度**: 案件保証なし（「獲得率93%」表記）、転職保証なし、リスキリング補助金が実質的な金銭保証

### C-05: COMPASS（ボディメイクコーチ養成スクール）

**URL**: https://compass-bodymake.com/

#### サービス概要
「ボディとマインドの両面から顧客の人生の転換点をつくる」コーチを育成。コーチングスキル＋ボディメイク理論＋ビジネス基礎の統合カリキュラム。受講生400名突破。

#### ターゲット顧客
- 筋トレ・ダイエット経験を活かしたい層
- 副業・独立志向者
- SNSに頼らない集客を望む人
- **基準**: 店舗実績・コンテスト履歴・トレーナー経験「不要」

#### 料金プラン
**公式LPは料金非公開**（個別相談誘導）。一括（銀振/クレカ）または分割（クレカのみ）対応。卒業生の口コミに「受講料の10倍を回収」とあり、推定数十万円〜100万円台か。

#### 主要機能・カリキュラム（4ステップ）
| ステップ | 期間 | 内容 |
|---|---|---|
| STEP1 | 入学〜1ヶ月 | 基礎習得（理論・コーチング・ビジネス基礎） |
| STEP2 | 2〜3ヶ月目 | 実践準備（セールス・コーチング） |
| STEP3 | 4ヶ月目〜 | 販売実践（モニター → 月収30万円目標） |
| STEP4 | 6ヶ月目〜 | 専門性深化（法人化・ブランド確立） |

#### 入会導線
```
無料動画視聴
   ↓
4ステップ動画コンテンツ
   ↓
ガイドブックDL（ポップアップ誘導）
   ↓
LINE登録
   ↓
個別相談・面談
   ↓
契約
```

#### サポート体制
- **オンライン**: 専用アプリ、認定コーチキャンプ、レベル制度、同期制度、朝活・夜活
- **オフライン**: 隔月交流会、2日間 CORE 対面研修、合宿（沖縄等）
- 保証制度: 明記なし

#### UI/UXの特徴
- 公式LPは「無料動画 → ガイドブック → LINE」の典型ローンチ型導線
- 受講生専用アプリの存在は言及（詳細不明）
- 卒業生実績（金額付き）を多数掲載

#### 強み・弱み
- **強み**: コンテンツ領域が筋肉塾と近い、卒業生実績の見せ方が上手い、コミュニティ＋合宿の体験設計
- **弱み**: 料金非公開（不透明感）、入会前に複数ステップを踏ませる動線（離脱多そう）

#### 軸C 追加項目
- **入会導線**: 動画 → LINE → 個別面談（典型的なローンチ型）
- **解約防止**: 同期制度・レベル制度・合宿でコミット感を高める
- **サポート**: アプリ・キャンプ・交流会・合宿の多層サポート
- **保証制度**: 明記なし（クチコミの「10倍回収」を実績訴求として代用）

### C-06: ダイエット卒業アカデミー（DMMオンラインサロン経由）

**URL**: https://lounge.dmm.com/detail/8585/

**主催**: 鳥巣愛佳氏（VIAS上野御徒町ボディメイクサロン代表）

#### サービス概要
食改善特化のオンラインサロン。月額980円のコミュニティで集客し、自社サロン（VIAS）の個別カウンセリング・パーソナルへ高単価誘導するモデル。

#### ターゲット顧客
- ダイエットを「卒業」したい一般層（女性中心と推測）
- リバウンドを繰り返している層
- 食事改善で痩せたい層

#### 料金プラン
| プラン | 価格 | 内容 |
|---|---|---|
| DMMサロン月額 | **月額980円** | コミュニティ参加 |
| 個別カウンセリング | 通常より割引 | 別途料金（推定数万〜十万円台） |
| VIAS パーソナル | 高単価帯 | 別途料金 |

→ **「低単価サロンで集客 → 高単価本サロンへ送客」型**

#### 主要機能・コンテンツ
- 限定ライブ配信（毎週月曜21:00）
- 有料コンテンツ配信（毎週火曜20:00）
- スレッド機能（メンバー投稿・悩み共有）
- 新規入会特典: 30分オンライン相談

#### 入会導線
```
DMMアカウント作成・ログイン
   ↓
サロンページで「入会」クリック
   ↓
月額決済
   ↓
入会後30分相談 → 個別カウンセリング誘導（VIAS本サロンへ）
```

#### サポート体制・コミュニティ
- DMMオンラインサロンのプラットフォーム機能を利用（独自アプリなし）
- スレッド形式の投稿
- 月1の限定ライブ配信

#### UI/UXの特徴
- **プラットフォーム借り型**（自社UI・自社認証なし、DMM依存）
- スレッド+配信のシンプルな構造
- 退会も DMM 側で完結（解約摩擦が低い＝ハードル低い）

#### 強み・弱み
- **強み**: 開発コストゼロ、低価格で間口広く、自社高単価サービスへの優れた送客導線
- **弱み**: DMM 依存（料率を取られる、ブランド主導権弱め）、退会容易、独自体験を作れない

#### 軸C 追加項目（重要観察）
- **入会導線**: DMMで完結（自社サイト不要）→ **超低摩擦**
- **解約防止**: ほぼなし（DMM側でいつでも解約可能）。代わりに「上位サービスへの誘導」で本命の収益化
- **サポート**: ライブ配信＋スレッドのみ、軽量
- **保証制度**: なし
- **示唆**: 高単価本体（VIAS）の集客装置として低単価サロンを置く構造。**筋肉塾のセミナーや無料コンテンツの位置付けと類似**
