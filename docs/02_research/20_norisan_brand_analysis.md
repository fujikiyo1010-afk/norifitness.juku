# 20 のり氏 既存ブランド分析

**作成日**: 2026-05-20
**目的**: のり氏の既存サイト（02_kinniku_lp / 01_tokuten/*）の **実際のカラー・タイポ・トーン**を抽出し、新サイトのデザインを **ブランド統一感のあるもの**に揃える根拠を提供する。
**調査対象**:
- `02_kinniku_lp/index.html`（筋肉塾 本体LP）
- `01_tokuten/carb-guide.html`（炭水化物の取扱説明書、2026年5月公開、最新作）
- `01_tokuten/ringosu-guide.html`（リンゴ酢×神食材6選、2026年5月公開、最新作）

→ 結果として、[19_design_spec.md](19_design_spec.md) のカラー提案は **方向性は合っていた**が、**実際のブランド色に揃えてアップデート可能**。

## 🔍 ブランド・アーキタイプ分析

### 「02_kinniku_lp」のキャラ
- **赤・黒・ティール**: 情熱と冷静の対比、ボディメイクLP らしい "煽り×信頼"
- **黄色マーカー**: 強調・要所
- **LINE緑 CTA**: 即時行動

### 「01_tokuten」のキャラ
- **オレンジ・グラデ**: 親しみ・親近感・温度
- **ティアリスト (SP/S/A/B/C/D/E/Z)**: ゲーミフィケーション気味（既に取り入れ済み）
- **キャラクター画像**: ヒーローに人物 or キャラを必ず配置
- **円形装飾**: 背景に光の輪をたびたび使用

### 共通する DNA
1. **ティールグリーン (#00897b)** → 信頼・冷静（両方で使用）
2. **LINE 緑 (#06C755)** → CTA 専用色（業界標準だが、のり氏ブランドの一部）
3. **深緑グレー (#1a3326 / #333)** → テキスト色、純黒は使わない
4. **黄色マーカー** → 強調手段（多用しない）
5. **角丸 8〜16px** → 親しみ感
6. **Noto Sans JP** → 特典LP標準（モダン・読みやすい）

**方向性は完全に正解だったが、ピンポイントで「のり氏 ブランド実色」に揃えると統一感が爆増する**。

### 新サイトへの推奨
**Noto Sans JP を採用**（理由）:
- 最新の特典LP（炭水化物・リンゴ酢、2026年5月）で採用
- Google Fonts で **CSS だけで読み込み可**、Cocoon等のテーマ依存ゼロ
- 多 weight 対応（400/500/700）
- モバイルでも美しく表示


## 🖼 ビジュアル要素の特徴

### ヒーローの作り方
- **キャラクター画像**（のり氏 or マスコット）を右側に配置
- 背景に**グラデーション + 円形装飾**（白の半透明円が浮く）
- タイトルは大きく、サブタイトルは控えめ

```
┌─────────────────────────────────┐
│  ┌─ badge ─┐         ⚪        │
│  │ LINE限定 │      ⚪          │
│  └─────────┘                    │
│                       ┌──────┐  │
│  炭水化物の          │      │  │
│  取扱説明書          │ キャラ │  │
│  ─────              │ ノリ氏 │  │
│  本気のあなたへ      │      │  │
│                       └──────┘  │
└─────────────────────────────────┘
```

### カードの作り方
- 白背景 + 薄オレンジボーダー (`#ffd0b0`)
- 角丸 **10〜16px**（中〜大、親しみ感）
- 影は薄め（`box-shadow:0 4px 14px rgba(0,0,0,.08)`）

### 内部セクションのアクセント
- 「黄色マーカー」で要所キャッチ
- 「色付きの引用枠」で重要発言
- 「Top 3 王冠装飾」「ティアバッジ (SP/S/A/B/C/D/E/Z)」← ゲーミフィケーションの先例
- 「絵文字 + バッジ + リボン」で達成感

## 🎙 コピー・トーンの実態

### 抽出したコピー（特典LP/carb-guide.html）

サンプル:
- 「**1,000人以上の指導**で見えてきた " 本当の選び方 "」← 実績訴求＋括弧強調
- 「**YouTube視聴者限定**の無料特典」← 限定感
- 「**ただ飲むだけ**では効果半減」← 一般通念の否定
- 「**論文ベースで徹底解説**」← 専門性
- 「**やめなかった人だけが**」← 努力と継続を讃える

### トーンの特徴
1. **括弧（"〜"）を多用** → キーフレーズの引き立て
2. **数字で実績** → 1,000人、●●%、論文●選
3. **「あなた」呼びかけ** → 距離感を縮める
4. **「本当の」「実は」「ただ●●では」** → 通念の否定からの提案
5. **絵文字は見出しに点在、本文は控えめ** → 過剰でない
6. **太字 + 色変化** → 重要点はビジュアルで強調

### 新サイトのコピー指針
- 学習ツールなので「煽り」は不要
- ただし「努力を讃える」「実績を語る」は継承
- "" 括弧、「あなた」呼びかけ、太字強調 は活用
- 絵文字は **見出しに1個** 程度、本文では避ける

→ 詳細は `21_microcopy_tone_guide.md` で展開（次の Step）

## 🎨 デザインアーキタイプ統合（新サイト向け）

新サイトのトーンを **「02_kinniku_lp（情熱）」と「01_tokuten（親しみ）」の中庸**に置く:

```
LP系: 赤×黒×ティール  ← 煽り強い（マーケ用）
                ↓
学習サイト: ティール×オレンジ ← ★ここ
                ↓
特典系: オレンジ×グラデ ← 親しみ強い（コンテンツ用）
```

→ つまり**「のり氏 LP の --lp-accent (#00897b) を Primary に昇格させ、特典LP の --g2 (#e65100) を Accent に統合」**することで、両方の世界観の良いとこ取りができる。
