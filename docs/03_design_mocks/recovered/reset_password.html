パスワードを設定してください

パスワードの条件
  • 8 文字以上
  • 英字 + 数字を含む
  • 2 つのフィールドが一致

画面の役割

  • パスワードリセットメール内のリンクから遷移
  • 新しいパスワードを入力 → 設定後ログインへ自動遷移
  • 戻るボタンなし (メール経由の直接遷移なので、ヘッダーシンプル)

構成 (上から)

  • ヘッダー: 「新しいパスワード設定」 (戻るなし)
  • アイコン (鍵 + 矢印) + 「新しいパスワード」+ 説明
  • 新パスワード + 確認 の 2 フィールド
  • パスワード条件チェックリスト (リアルタイム反映)
    • 8 文字以上 (緑チェック = 満たす)
    • 英字 + 数字を含む
    • 2 つのフィールドが一致
  • 「パスワードを変更する」ボタン

動線

  • 変更成功 → /login に自動遷移 (or 完了トースト → ログイン)
  • 条件不一致 → ボタン非活性 + 該当ルールが赤表示
