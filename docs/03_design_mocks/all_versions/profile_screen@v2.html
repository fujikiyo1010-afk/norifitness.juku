🗺 画面の役割

  • ヘッダー右上の 👤 アイコンから開く
  • 下部タブからの遷移ではなく、各画面のヘッダーから呼び出す
  • 受講生自身の情報 + 学習統計の表示

📐 構成 (上から)

  • ヒーロー: アバター + 名前 + 入会日 (E案グラデ背景)
  • 学習統計: 3 セル (累計学習日数 / 視聴時間 / 完了レッスン) ・ コース完了画面のサマリと整合
  • コース進捗: 5 コース一覧、各々進捗バー
    • 完了済 = ✓ アイコン + バー深緑 100%
    • 進行中 = 📚 アイコン + バー緑
    • 未着手 = 📖 アイコン + バー 0%
  • 最近の活動: 直近 5 件 (体組成記録 / 月次添削 / レッスン視聴 / ツール使用)
  • 編集 CTA: 「プロフィールを編集する」(ティール緑、角丸 14px)

📊 学習統計の数値

  • 累計学習日数: アプリにログイン or レッスン視聴があった日数
  • 視聴時間: 全レッスンの動画視聴時間 (累計)
  • 完了レッスン: 「できた!」を押したレッスン数 / 全 217
  • ※ コース完了画面のサマリと同じ指標 (ポジティブ指標のみ)

🔄 最近の活動の元データ

  • body_metrics テーブル → 体組成記録
  • monthly_audits テーブル → 月次添削
  • レッスン視聴履歴 → レッスン名 (時系列)
  • tool_calculations テーブル → ツール使用 (時系列)
  • 5 件取得 → 1 つのストリームに統合表示

✏️ 編集できる項目

  • 編集ボタン → 編集モード or 別画面
  • 編集可能: 名前 / アバター画像
  • 編集不可: 入会日 (システム情報)
  • ※ メールアドレス・パスワードは設定画面側で編集

🎨 デザイン方針

  • ヒーロー帯は E案グラデ (温かさ、プロフィール = 個人の領域)
  • アバターは円形、ティール緑のイニシャル表示 (画像未設定時)
  • コース進捗バーで「自分の旅路」を可視化
  • カード角丸 12px
