# 管理画面 アラートタグ 仕様

**作成日**: 2026-06-11
**目的**: 管理画面で受講生の状態を瞬時に把握するためのアラートバッジ仕様

トレーナークラウド管理画面の「体重逆行 / 体重途絶 / 食事途絶 / 目標から乖離」等を参考に、 のりfitness 流に再設計。

---

## 設計原則

- **緊急度 2 段階**: 🔴 緊急 (urgent) / 🟠 注意 (warn)
- **表示場所 3 箇所**:
  - ホームダッシュボード (今すぐ対応 / 今日中に確認 リスト)
  - 受講生一覧 (`/admin/users` テーブルの状態列)
  - 受講生ハブ画面 (`/admin/users/[id]` 概要タブ)
- **自動判定**: バッチで毎日 0:00 (or ログイン時) に集計
- **閾値はコードに直書き** → Phase 4 で **設定画面から変更可能** に拡張予定

## 表示ロジック

### ホームダッシュボード
- 🔴 緊急 タグ 1 個以上 → 「今すぐ対応」セクション
- 🟠 注意 タグ 1 個以上 (緊急 0) → 「今日中に確認」セクション
- アラート 0 → 表示しない (= サイレント)

### 受講生一覧
- 状態列にバッジ最大 2 個 + 「+N」 (緊急度順)
- フィルタ「要対応」= 緊急タグありで絞り込み

### 受講生ハブ (概要タブ)
- 全アラートを縦並びで列挙、 各タグに「対応する」リンク (該当画面へ遷移)
