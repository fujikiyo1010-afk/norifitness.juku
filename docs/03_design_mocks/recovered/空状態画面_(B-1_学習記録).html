🗺 画面の役割

  • 受講生 UI でコンテンツがまだない時の表示
  • 「何もない」を冷たくせず、次の行動へ背中を押す
  • キャラが「いるだけで温かい」雰囲気を作る

📐 構成 (上から)

  • ヘッダー: 戻る + ページタイトル (通常画面と同じ)
  • 本体: 中央配置でキャラ + タイトル + メッセージ + CTA
  • 背景: 装飾なし、シンプル (起動画面と差別化)
  • キャラ: 110px、起動より小さめ (主役じゃない、添える存在)
  • CTA: ティール緑、角丸 14px (機能ボタンと同じ)

🎬 アニメーション

  • キャラ: 0.0s 〜 0.6s フェードイン
  • タイトル: 0.2s 〜 0.6s フェードイン
  • メッセージ: 0.3s 〜 0.7s フェードイン
  • CTA: 0.4s 〜 0.8s フェードイン
  • (順次表示で「画面に温度感が宿る」演出)

📝 メッセージのトーン (筋肉塾標準)

  • 「ありません」を冷たくせず、続けて行動への誘いをセット
  • 命令形を避ける (「◯◯してください」より「◯◯してみましょう」)
  • 絵文字は CTA ボタンのみ (本文は文字だけで雰囲気を作る)
  • 「みなさん」「あなた」は使わず、シンプルな言い切り
