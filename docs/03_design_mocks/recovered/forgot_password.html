画面の役割

  • 受講生がパスワードを忘れた時の再設定動線の入口
  • メアド入力 → Supabase Auth がリセットメール送信
  • メール内のリンクから /reset-password (新規パスワード設定画面) に飛ぶ

構成 (上から)

  • ヘッダー: ← 戻る + 「パスワードを忘れた方」
  • アイコン (鍵) + 「パスワード再設定」+ 説明
  • メールアドレス入力欄
  • 「リセットメールを送信」ボタン
  • 注意書き (迷惑メール / LINE サポート)
  • 「ログインに戻る」リンク

動線

  • メール送信成功 → 完了表示 (送信しました画面 or トースト) → ログインに戻る
  • 戻る → /login
