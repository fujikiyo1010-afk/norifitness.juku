🔄 2026-05-24 改修版

① 性別ボタンを 3 択 (男性 / 女性 / その他) に拡張
② デフォルト 未選択 (どれも selected なし)
③ 「その他」選択時のみ、真下に 計算式選択の但し書き が動的表示
④ ツール群配色を インディゴ #3949ab に変更
⑤ 「目標管理シートに反映」ボタンだけ ティール緑 #00897b 維持 (目標シートとの繋がり)

🗺 画面の役割

  • ツール一覧 → 「① 体脂肪率計算」タップ → このページ
  • 男女両対応 + 「その他」配慮 (3 択)
  • 計算結果 → 目標管理シート ① 現状把握 + 体組成記録 に自動転記

📐 構成 (上から)

  • ヒーロー帯: ツール番号 + タイトル + 説明 (E案グラデを インディゴ系に調整)
  • 入力カード:
    • 性別 3 ボタン (デフォルト未選択)
    • 「その他」選択時 → 但し書き + 男性式/女性式ボタンが動的に追加表示
    • 身長・ウエスト・首回り (必須)
    • ヒップ (女性式選択時のみ動的表示)
    • 体重 (任意、体脂肪量・除脂肪量も計算)
  • 計算ボタン: インディゴ + 角丸 3px (シャープ、ツール内アクション)
  • 結果カード: 大表示 42px Inter (体脂肪率) + サブ 2 セル
  • 計算式: 透明性のため公式を明記
  • 反映エリア: ☑ 体組成記録にも反映 + 目標シート反映ボタン (ティール緑 + 3px)

🎨 配色ルール (2026-05-24 確定)

  • ツール群メイン: インディゴ #3949ab (計算・分析の色)
  • 目標シート反映ボタン: ティール緑 #00897b (他システムへの転送 = 目標シートの色)
  • 結果カード薄黄背景はそのまま (嬉しい情報のニュアンス)
  • 意味分担: ツール = インディゴ / 受講生 UI 全体 = ティール緑 / 添削 = 薄黄

🔄 動的フォーム挙動

  • 性別「男性」選択 → ヒップ欄を非表示、男性式で計算
  • 性別「女性」選択 → ヒップ欄を表示、女性式で計算
  • 性別「その他」選択 → 但し書き表示 + 男性式/女性式の選択ボタンを表示
    • 男性式を選択 → ヒップ欄を非表示、男性式で計算
    • 女性式を選択 → ヒップ欄を表示、女性式で計算
  • 性別未選択で計算ボタン押下 → 「性別を選んでください」エラー

🧮 計算ロジック

  • 男性式: BF% = 495 / (1.0324 − 0.19077 × log10(W−N) + 0.15456 × log10(H)) − 450
  • 女性式: BF% = 495 / (1.29579 − 0.35004 × log10(W+Hip−N) + 0.22100 × log10(H)) − 450
  • 体脂肪量 (kg) = 体重 × BF% / 100
  • 除脂肪量 (kg) = 体重 − 体脂肪量
