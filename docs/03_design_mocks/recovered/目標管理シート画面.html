🗺 画面構成

  • ヘッダー: 戻る← / 目標管理シート / ⋯ (履歴メニュー)
  • 進捗バー: 6 セクション中の記入数を表示
  • セクション 6 枚: 各「★必須」と「✓記入済/未記入」マーク
  • のり社長コメント: 初回提出後に表示、動画リンク付き
  • ボトムリンク: 編集履歴 / 変化を見る
  • 保存バー: 下書き保存 + 保存する (明示保存式)

🛠 ツール連動

  • ① 必要カロリー計算 → BMR / メンテkcal 反映
  • ② 減量期間逆算 → 目標体重・週ペース・到達日 反映
  • ③ (週間トレ強度はツール反映の他、③ 内で直接編集も可)
  • ④ PFC・カーボサイクル → kcal/PFC/週糖質 反映
  • ツール反映済フィールドは readonly + パステル背景で区別

✅ 反映した合意事項

  • 全項目必須（自己管理目的）
  • 体脂肪率は男女両対応 (アメリカ海軍式・性別自動切替)
  • 性別/身長/年齢は プロフィール連動 (自動反映、編集不可)
  • ツール作業値 ≠ シート連動値、明示「反映」 (③で議論済)
  • のり社長コメント = 動画 + テキスト、初回必須
  • 明示保存式 (下書き保存 + 保存する)
  • 編集履歴・変化を見るへの導線

🎨 デザイン原則

  • 明るい緑 #4ECD5D + 深緑 #1f6b3e
  • Lucide outline 線画黒一色アイコン
  • 必須マーク ★ は赤、セクション完了 ✓ は緑
  • のり社長コメントだけ薄オレンジ系で差別化 (温度を出す)

⚠️ 未確定 / 検討点

  • セクション ⑥ 8 項目: 実際には全 8 項目縦並び、長くなる (展開/折りたたみ機能を入れるか)
  • 下書き保存: localStorage か DB か（中断 OK の方針なので DB が筋）
  • セクション順序: 1→2→3→4→5→6 の順で誘導するか、自由順か
  • 編集履歴: 表示形式 (時系列リスト / Diff)
  • 初回未提出時: のり社長コメント欄は非表示 or 「のり社長コメント待ち」表示
