# docs/ — 筋肉塾 新サイト構築 ドキュメント

フェーズ0〜4で生成される抽出物・設計書・調査結果の置き場。

---

## 📁 構成

```
docs/
├── README.md            ← このファイル
├── 00_premises/         ← フェーズ0 前提条件抽出物
│   └── _toc_map.md      ← 既存ナレッジの目次マップ
└── 01_expa/             ← エキスパ調査結果
    └── screenshots/     ← 画面ベース調査時のスクショ（D方式時に使用）
```

フェーズ進行に応じて追加予定:
- `02_design/` — フェーズ2 デザイン関連（参考サイト、世界観、ワイヤー、カンプ）
- `03_implementation/` — フェーズ3 実装関連（Supabase スキーマ、API、テスト）
- `04_operations/` — フェーズ4 運用関連（移行手順、エキスパ閉鎖、運用フロー）
- `_summary_for_president/` — 社長確認用サマリー（フェーズ0完了時必須）

---

## 📜 保存ルール

- 抽出結果はすべて `.md` 形式で保存
- 各ファイル冒頭に**抽出元**を明記（例: `Source: ../../03_brain/memory/noriAI_business.md`）
- 「未記載」「※推測」「※要確認」を明示
- 筋肉塾特化のものはここに、汎用ナレッジは `../../03_brain/memory/` にも逆輸入
