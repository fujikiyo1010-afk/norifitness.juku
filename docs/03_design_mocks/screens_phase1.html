② コース一覧の構成要素

  • ヘッダー: 戻る← / 「コース」/ ヘッダー右上 ✉（赤ドット付き）
  • 検索バー: コース・章を検索（タップで検索画面 or インライン絞り込み）
  • フィルタタブ: すべて / 進行中 / 未着手 / 完了 — 数字付き、横スクロール可能
  • コースカード横並び: サムネ + タイトル + 章/レッスン数 + 進捗バー + 「進行中 45% (36/79)」明示
  • 状態色分け: 未着手はグレー文字、進行中は緑、完了は ✓ 表示

③ コース詳細の構成要素

  • ヒーロー: 緑グラデ背景 + 大サムネ + タイトル + 進捗バー + 「続きから」白 CTA
  • 章ロウ: 章番号バッジ（完了は緑塗り、進行中は薄緑、未着手は淡） + 章名 + 進捗
  • 章を開くとレッスン一覧表示:
    • ✓ 完了 (緑丸)
    • ▶ 続きから (黄丸 + 「続きから」黄バッジ)
    • ○ 未着手 (グレー枠)
  • 第 3 章だけ開いた状態を表示（実際はタップで開閉、他の章は閉じた状態）

④ レッスン詳細の構成要素

  • ヘッダー: 戻る← / 「レッスン」/ 右上 ⋯（ブックマーク / 共有等の予備領域）
  • パンくず: 「限定ボディメイク > 第 3 章 > L4」で現在地を明示
  • 動画プレイヤー: 16:9、Vimeo 埋め込み、再生バー + 時間表示
  • レッスン情報: タイトル + 講師（のり社長アバター）+ 時間
  • できた! ボタン: LINE 緑 #06c755 + 太く目立つ + ✓ アイコン付き
  • アコーディオン 2 つ: 振り返り（任意）/ 配布資料（件数表示）
  • 次のレッスンへ: 薄緑カードで自然な誘導（自動再生はしない）

⚠️ 未確定 / 検討点

  • フィルタタブの設計: 「すべて/進行中/未着手/完了」で OK か、カテゴリ別（食事/筋トレ等）が良いか
  • コース詳細のヒーロー: 緑グラデ強めだが、これは "中庸寄り" の温度。落ち着かせるなら白背景+線の選択肢もあり
  • 章ロウのアコーディオン: デフォルト全閉 or 進行中の章だけ開く（モックは後者を採用）
  • レッスン詳細の右上 ⋯ メニュー: ブックマーク（Phase 4）等の予備領域として確保。今は機能なし
  • 動画下に "完了マーク" を出すか / "できた!" ボタン押した後の見た目（押下後のロック状態）
