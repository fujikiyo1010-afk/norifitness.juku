🗺 画面の役割

  • ホーム画面 → 機能ブロック「ツール」をタップ → このページに遷移
  • 4 ツールを順序固定で並べる (番号付きカード)
  • タップで各ツールの個別画面へ

📐 構成

  • ヒーロー帯: E案グラデ (温かい) + タイトル「🛠 計算ツール」
  • ツールカード: 番号 + ツール名 + 短い説明 + →
  • flow-tip: 「はじめての方へ」順序ガイダンス

🎨 デザイン方針 (2026-05-24 更新)

  • ツール群 = インディゴ #3949ab (Material 600、ティール緑と同明度)
  • 役割色分け: 受講生 UI 全体 = ティール緑、ツール = インディゴ、添削 = 薄黄
  • ツール画面に入ると色が変わるので「ここは計算する場所」が直感的に分かる
  • ヒーロー帯のグラデも E案 (ティール) から インディゴ系 (#e8eaf6 → #fffbe6) に微調整
  • カード角丸 14px (一般カードに揃える)
  • 計算ボタン・反映ボタンは個別ツール画面で 3px (シャープ) 適用

📋 採用したツール 4 つ

  • ① 体脂肪率計算 (特典 25 ベース、男女両対応 ← 拡張)
  • ② 必要カロリー計算 (特典 23 ベース)
  • ③ 減量期間逆算 (特典 24 ベース)
  • ④ PFC・カーボサイクル設定 (特典 26 ベース、STEP 1+2 統合)

🔄 目標管理シート連携

  • 全 4 ツールに「目標管理シートに反映 →」ボタンを配置
  • ① は body_metrics (体組成記録) にも自動転記オプションあり
  • ④ のカーボサイクル配分 = 条件付き + アコーディオン表示 (合意済)
  • ② の反映時 = 案 γ ハイブリッド (推奨マーク付き選択ダイアログ、合意済)
