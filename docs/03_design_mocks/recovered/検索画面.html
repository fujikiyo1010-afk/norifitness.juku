📐 構成 (上から)

  • ヘッダー: シンプル「検索」
  • 検索ボックス: フォーカス済み (画面遷移時に自動フォーカス)
  • 検索前: 最近の検索 (5 件) + よく検索される (5 件、運営側設定)
  • 検索後: 結果リスト (コース / 章 / レッスン の 3 グループ)
  • ※ モック上は両方併記、実装時は検索状態で切り替え

🎯 検索結果の構造

  • 3 グループ: コース / 章 / レッスン (上から重要度順)
  • 各カードに パンくず (どのコース・章に属するか)
  • レッスンには スニペット (本文の抜粋)
  • 検索語は 黄色マーカー でハイライト (添削色と同じ色)

🔄 動的挙動

  • 入力ボックスが空 → 「最近の検索」+「よく検索される」を表示
  • 入力開始 → リアルタイムで結果リスト表示 (デバウンス 300ms)
  • 結果ゼロ → empty_state B-4 (作成済) に切り替え
  • 履歴の ✕ で個別削除

🎨 デザイン方針

  • シンプル、リスト形式
  • カード角丸 12px (一般カードと同等)
  • 検索ボックスは focus 時に枠色変更 (ティール緑)
  • 結果カードの hover で枠色変更
