画面の役割

  • 受講生のアプリ入口
  • 入会後 (アカウント手動発行) の通常ログイン
  • 初回 / リピートとも同じ画面 (中で分岐)

構成 (上から)

  • キャラ (90px、起動より小さめ) + 筋肉塾ロゴ + タグライン
  • メールアドレス + パスワード フォーム
  • 「ログイン」ボタン (ティール緑)
  • 「パスワードを忘れた方」リンク → /forgot-password
  • 区切り線
  • 「招待コードで新規登録する →」リンク → /invite (グレー、控えめ)

動線

  • ログイン成功 → / (ホーム or オンボ判定)
  • パスワード忘れ → /forgot-password
  • 新規登録 → /invite
