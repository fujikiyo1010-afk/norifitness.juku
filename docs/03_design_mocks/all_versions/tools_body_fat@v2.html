🗺 画面の役割

  • ツール一覧 → 「① 体脂肪率計算」タップ → このページ
  • 男女両対応 (特典 25 は男性式のみだったが、女性式を追加)
  • 計算結果 → 目標管理シート ① 現状把握 + 体組成記録 に自動転記

📐 構成 (上から)

  • ヒーロー帯: ツール番号 + タイトル + 説明 (E案グラデ)
  • 入力カード: 性別 → 身長・ウエスト・首回り → ヒップ (女性のみ動的表示) → 体重 (任意)
  • 計算ボタン: ティール緑、角丸 3px (シャープ、ツール専用ルール)
  • 結果カード: 大表示 (体脂肪率 42px) + サブ 2 セル (体脂肪量 / 除脂肪量)
  • 計算式: 透明性のため、使った公式を明記 (Inter monospace)
  • 反映エリア: ☑ 体組成記録にも反映 (default ON) + 目標シート反映ボタン (黒)

🧮 計算ロジック

  • 男性: BF% = 495 / (1.0324 − 0.19077 × log10(W−N) + 0.15456 × log10(H)) − 450
  • 女性: BF% = 495 / (1.29579 − 0.35004 × log10(W+Hip−N) + 0.22100 × log10(H)) − 450
  • 体脂肪量 (kg) = 体重 × BF% / 100
  • 除脂肪量 (kg) = 体重 − 体脂肪量

🔄 動的フォーム挙動

  • 性別「男性」選択 → ヒップ欄を非表示
  • 性別「女性」選択 → ヒップ欄を表示 (女性式は 4 項目必要)
  • 体重は 任意入力: 入れると体脂肪量・除脂肪量も計算、入れなくても BF% は出る

🎨 デザイン適用

  • カード角丸 = 14px (一般カード)
  • 計算ボタン角丸 = 3px (ツール専用、シャープ)
  • 目標シート反映ボタン = 3px + 黒 (主要 CTA は黒、Phase 2-7 標準)
  • 結果カード = 薄黄背景 (添削色と同じ、嬉しい情報のニュアンス)

📤 反映ボタンの挙動

  • 「目標管理シートに反映する」押下:
    • 目標シート ① 現状把握「体脂肪率」に書き込み
    • ☑ チェック ON なら body_metrics の最新行にも転記
    • 反映後 → 目標シート閲覧モード (overview) に遷移、該当フィールドにハイライト
