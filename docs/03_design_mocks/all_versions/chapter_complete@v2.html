📝 メッセージのトーン

  • 過剰に褒めない ("すごい!" "天才!" NG)
  • 事実を肯定 ("お疲れさまでした" "終えました")
  • 次への誘い はソフト ("次の章へ進む" 程度)
  • 絵文字は CTA に含めない (本文は文字だけ)

🎨 デザイン方針

  • 背景: E案グラデ (温かい、起動画面と統一)
  • 装飾円で深さ感、コース完了より控えめ
  • 紙吹雪なし (コース完了との差別化)
  • キャラ通常ポーズ (祝福専用ポーズは Phase 2)

🔄 閉じ方

  • [次の章へ進む →] → モーダル閉じる + 次の章のレッスン一覧に遷移
