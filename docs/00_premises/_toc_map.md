## 📌 重要な現況コンテキスト（2026-05-18 時点）

抽出時に**必ず意識すべき背景**:

1. **経営危機モード**: 借入¥10.49M、現金ほぼゼロ。5月内に売上を作らねばならない
2. **5/20 動画ローンチ**（6,000人想定）+ **5/23 セミナー**（12,000人想定）の二段ロケット進行中
3. セミナー価格 **¥298K〜¥327K**、個別面談で 80% 成約目標
4. 筋肉塾本体は **月商¥750K 目標 / 現状13人 → 22人達成が最大課題**
5. AI（noriAI）導入が経営判断としても急務

### 本プロジェクト（新サイト構築）への示唆
- 新サイトは単なる技術移行ではなく**収益拡大の道具**
- MVP 範囲は「**売上に直接効く機能**」を最優先
- 例: スムーズな入会導線、決済の摩擦削減、課金率向上、解約防止 UX、アップセル設計
- 完璧主義より速度（経営状況的に長期プロジェクトは負担）
