🗺 全体構成

  • PC画面想定 (管理画面はスマホ非対応で OK の合意あり、スマホでも一覧チェック程度は可能)
  • ヘッダー: ブランド + ログイン中ユーザー
  • 検索バー: 受講生名で絞り込み
  • フィルタタブ: 全て / 危険信号 / 未返答 / 返答済
  • リスト: 3 セクション (危険信号 → 未返答 → 返答済)、古い順固定

📍 シーン別の使い方

  • シーン 1: 月初の集中作業 — 「未返答」タブ → 古い順に上から処理 → 行タップで個別作業画面 (3b) へ
  • シーン 2: 隙間時間の確認 — 件数バッジで「未返答 12 件」「危険信号 2 件」を一目把握
  • シーン 3: 危険信号対応 — 赤バッジで最上段固定、優先処理

🎨 デザインルール

  • 危険信号: 赤 #d32f2f (要所のみ、責める色ではあるが運営側用なので OK)
  • 未返答: 黄 #b8860b (受講生 UI と同じ "添削色")
  • 返答済: ティール緑 + 50% 透明度 (完了感)
  • 角丸 8-12px: 受講生 UI (14px) より少しシャープ (情報密度高い)
  • フォント: 数値・日付 = Inter、テキスト = Noto Sans JP

⚙ 機能仕様

  • 並び順: 古い順 (経過日数の大きい順)
  • 件数バッジ: 各タブ右に未読数表示
  • クリック: 行タップで画面 3b へ遷移
  • 危険信号判定: submitted_at から 7 日経過 (合意済仕様)

⚠️ 未確定 / Phase 2 候補

  • 受講生名タップで 個別ダッシュボード (画面 3c) ポップアップ — Phase 2 候補
