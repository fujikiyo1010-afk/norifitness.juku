## 🎯 設計原則（v2.1 で再確認）

```
🥇 学習ツールであること（絶対）
🥈 使いやすさ・満足度・実施完工率
🥉 ゲーミフィケーション（"学ぶ動機の補強" のみ、邪魔しない範囲）
```

→ **ゲーミフィケーションが学びの邪魔をしない**。バッジ・連続記録・進捗バーは「学んだ証として残る」もので、「取りに行く目標」になってはいけない。

---

## 🔄 v2 からの主な変更点

### 抑制した要素
| 変更 | 内容 |
|---|---|
| ❌ **XP表示の全削除** | 「+10 XP獲得！」等のゲーム用語を全廃 |
| ❌ **レベル名（達人/師範/グランドマスター）廃止** | チャラさが学習ツールのトーンとズレる |
| ❌ **コメントに👍 = XP** 廃止 | いいね稼ぎコメントの誘発を防止 |
| ❌ **キャラ通知** 削減 | 「のり氏が待ってます…」型は控えめに |
| 🔽 **HOMEダッシュボード 4枠 → 2枠**に縮小 | 学習指標（視聴時間・できた！数）に集中 |
| 🔽 **🔥連続記録** ヘッダー固定 → **目立たない位置**に | プレッシャー軽減 |
| 🔽 **バッジ 20〜50個 → 10〜15個**に絞る | 価値希薄化を防ぐ |

### 統合した画期的「学びを深める」5機能
| 新機能 | 配置場所 | 学習価値 |
|---|---|---|
| 📝 **3行 振り返り** | レッスン詳細画面 | Feynman技法、自分の言葉で要約 |
| 🔖 **動画の瞬間スクラップ** | レッスン詳細 + マイページ | 学びのハイライト保存 |
| 💪 **実生活への落とし込み** | レッスン詳細（目標シート連動） | 学習 → 行動の橋渡し |
| 🌟 **先輩からの一言** | レッスン詳細 | 卒業生の実体験、社会的学習 |
| 🔁 **逆向き学習モード** | 章一覧 | 試験を先に → 苦手を狙って学ぶ |

## ゲーミフィケーション要素の方針（v2.1）

| 要素 | v2 | v2.1 | 理由 |
|---|---|---|---|
| XP 表示 | あり（「+10 XP」等） | **廃止** | 学びの言葉から逸脱 |
| レベル名 | 入塾生〜グランドマスター | **廃止** | のり氏トーンとズレ |
| バッジ | 20〜50個 | **12個に絞る** | 価値希薄化を防ぐ |
| Streak (連続記録) | ヘッダー常時固定 | **HOME末尾に小さく** | プレッシャー軽減 |
| 紙吹雪演出 | できた！毎回 | **章完了・試験合格のみ** | 重要節目限定 |
| 👍 で +XP | 加点ルール | **加点なし、純粋な反応** | いいね稼ぎ防止 |
| ランキング | リーグ機能候補 | **不採用** | 比較ストレス回避 |
| ダッシュボード | 4枠（XP含む） | **2枠（学習指標のみ）** | 学習集中 |
