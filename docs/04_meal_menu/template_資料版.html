A. 鶏むねチャーシュー（むね1.5kg分）

材料：鶏むね1.5kg／しょうゆ大3／酒大3／おろし生姜小2／おろしにんにく小2／黒こしょう・一味 少々／白ねぎ青い部分
  1. むね肉をフォークで刺し、厚い所は開いて厚みを均一に。
  2. ポリ袋に調味料・ねぎ・肉を入れ、もんで30分置く。
  3. 沸騰した湯を火から下ろし、袋ごと沈めて蓋・余熱で40〜45分。
  4. 薄切りにして1食120〜200gに小分け（冷蔵3日／冷凍2週）。

B. 低脂質スパイスカレーの具（4〜5食分）

材料：鶏むね600g／玉ねぎ2／なす2／ピーマン2／トマト缶1／クミン・コリアンダー・ターメリック 各小2／塩小1／生姜・にんにく 各小1
  1. 鶏むねと野菜を一口大に切る。
  2. 玉ねぎを飴色近くまで炒め、生姜・にんにく・スパイスを30秒炒める。
  3. 鶏→野菜→トマト缶→水200mlの順で入れ、塩で調え15分煮る。
  4. 1食ずつ小分けして冷凍。食べる時はレンジ加熱。
