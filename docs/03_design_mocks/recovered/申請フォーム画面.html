画面の役割

  • 受講生主導の新規入会動線の起点
  • 振込確認後、きよむさんが LINE で送るリンクから開かれる
  • 受講生が自分の氏名・メアドを入力 → きよむさん承認 → アカウント有効化リンクがメールで届く

構成 (上から)

  • ヘッダー: 「入会申請」(戻るボタンなし、 LINE から直接遷移)
  • アイコン (書類) + 「筋肉塾へようこそ」 + 説明
  • フォーム 2 項目:
    • 氏名 (LINE と同じ名前を推奨、ヒント明示)
    • メールアドレス (有効化リンク送信先、ヒント明示)
  • 「申請する」ボタン
  • 注意書き (1 営業日以内に確認、有効化リンクを送付)

確定方針 (2026-06-10)

B-2 マジックリンク方式
「招待コード」は廃止、メアド+リンクで完結する設計に統一。

動線:
1. 受講生が支払い (銀行振込 or Stripe) → きよむさん入金確認
2. きよむさんが LINE で申請フォームのリンク送付
3. 受講生が この画面 で氏名 + メアド入力
4. 管理画面の「申請中」リストに表示 → きよむさん承認
5. システムがメールで「アカウント有効化リンク」自動送信 (Resend 経由)
6. 受講生がリンククリック → パスワード設定画面 → ホーム → オンボーディング 8 ステップ
