## 3. DBテーブル（✅40テーブル）※改修に効くものを抜粋、全一覧は別掲

**体組成/目標/ツール**
- `body_metrics`：user_id / recorded_at(date) / weight_kg(numeric4,1) / body_fat_percent(3,1) / waist_cm(4,1) / note。**UNIQUE(user_id,recorded_at)**・RLS本人のみ・**写真列なし**。
- `goal_sheets`：user_id(PK) / content(jsonb・中に`goal_selection.target_weight_kg`) / reviewed_by→admin_users / last_review_requested_at。
- `tool_calculations`：4ツールの計算結果保存(body_fat/calorie/diet_period/pfc_carb)。

**学習/コース**：courses / chapters / lessons(vimeo_url,content_json) / lesson_progress / lesson_reviews / comments / real_world_actions(実践リスト) / exams系(新)・tests系(旧・非推奨)。

**筋トレ**：workout_templates / user_workout_carte / user_workout_menu(cycles,effective_from,is_current) / user_workout_request / user_carte_request。**実施ログ用テーブルは無し**。

**添削/連絡**：monthly_audits(提出→のり動画返信の4状態) / conversations+messages(チャット・Realtime) / notifications / broadcast_notifications / announcements。

**運用**：users / user_profiles / admin_users / shipments / invitations / signup_requests / push_subscriptions / reminder_log / login_attempts / stripe_events / user_status_history / faqs / video_library。

**ヘルパー**：`is_admin()` / `is_superadmin()` / `set_updated_at()` ほか。
**非推奨(残置)**：`tests`/`test_attempts`(→exams系に置換)。`security_events`は作成後 即drop済。
⚠️ `video_library`/`login_attempts` は**本番へSQL Editor手動適用**(CLIマイグレーション運用ではない)。
