# 03 深掘り: COMPASS（ボディメイクコーチ養成スクール）

**URL**: https://compass-bodymake.com/
**価格**: **非公開**（説明会で提示、卒業生証言から推定で数十万〜100万円台）
**販売形態**: 動画ファネル + LINE + 個別説明会
**筋肉塾との関係**: コンテンツ領域（ボディメイク）が最も近い、コミュニティ・卒業生実績の見せ方の参考

```

   ↓
```

## 6. 筋肉塾新サイト設計に活かせるポイント ⭐

### ① 4ステップ動画ファネル
- 1本目で価値提供
- 2〜4本目で段階的に教育
- 4本目最後で「次の一歩 = ガイド or LINE」を提示
- **筋肉塾**: のり氏の無料コンテンツも段階的シーケンス化できる

### ② マッチョタイプ診断 = 「自分ごと化ツール」
- 簡易診断で「自分の現状」を可視化
- 診断結果に「あなたに合うコース」を提示
- **筋肉塾**: 「あなたのカラダタイプ診断」「目標タイプ診断」をLPに置く

### ③ LINE登録 = メアドの次の階段
- メアドはオフィシャル、LINEはカジュアル
- LINEは開封率が高い（メールの数倍）
- **筋肉塾**: メアド → LINE の2段階リスト化を検討

### ④ 説明会で料金提示 = 価値理解後のクロージング
- ホームページに料金を載せない理由 = 「価格 vs 価値」の認識ギャップ防止
- 説明会で個別対応 → 「自分にとっての価値」を理解させてから提示
- ただし**透明性を求める層は離脱**するトレードオフあり
- **筋肉塾**: のり氏のブランド力次第。透明（明示）vs 非透明（説明会）の戦略選択

### ⑤ 同期制度 + 合宿 = コミット感醸成
- 6ヶ月の同期グループ
- 2日間対面研修・沖縄合宿
- 「一緒に頑張る仲間」が強烈な定着剤
- **筋肉塾**: 期生制（◯期生）・オフ会・合宿の検討。完全オンラインでも「期」の概念は持てる

### ⑥ 「永久滞在制度」 = 卒業後の囲い込み
- 卒業後も交流会・キャンプに参加可
- 卒業生 = 口コミ発信者 = 営業マン化
- **筋肉塾**: アルムナイ会員制度・継続コミュニティ

### ⑦ 卒業生実績の「金額表示」
- 「介護職→売上16万円」「副業志向→売上54万円」
- 具体的な数字 + 元の属性で「自分にもできそう」感
- **筋肉塾**: 「ビフォーアフター + 数字（kg/cm/期間）」を一覧化

## 7. 避けるべきアンチパターン

### ❌ 料金非公開の透明性問題
- LPに価格がない = ググって価格を探す動線が生まれる
- 「怪しい」「いくらか不安」と感じる層は離脱
- 説明会前提 = LPだけでは買えない（コンバージョン遅い）
- **筋肉塾**: 価格は明示する方向で。透明性 > 非透明性

### ❌ 入会まで多ステップ（動画4本→LINE→説明会）
- 離脱ポイントが多すぎる
- 「すぐ買いたい」層を取り逃す
- **筋肉塾**: 「すぐ買いたい人用の直接申込ボタン」を別途置く

### ❌ 「説明会 = セールス」のプレッシャー
- 高単価セールスの典型構造
- 押し売り感を嫌う層は離脱
- **筋肉塾**: のり氏のキャラ・ブランドで「教育＋共感」型のクロージングが可能

### ❌ 卒業後の囲い込みが「依存」に転化するリスク
- 永久滞在 = COMPASS が無いと不安になる
- 「自立を支援する」と言いながら依存させる構造矛盾
- **筋肉塾**: 「卒業 = 一人立ち」を建前にしつつ、アルムナイは選択肢として残す
