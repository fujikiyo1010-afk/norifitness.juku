🗺 画面の役割

  • 5 コース 22 章 217 レッスン全部終えた瞬間に表示
  • 一生に 1 回しか出ない画面、最大級の特別感
  • 「責めない / 強制しない / 煽らない」原則は守りつつ 「静かに、でも特別に」

📐 1 コース完了画面との差分

  • 紙吹雪 20 粒 → 40 粒 (2 倍)
  • 「修了証カード」追加 (この画面の主役)
  • サマリは 5 コース通算 (189 日 / 42.8 時間 / 217 レッスン)
  • SNS シェア 3 ボタン追加 (X / 画像保存 / リンクコピー)
  • 「次のコースへ」CTA なし → 「ホームに戻る」のみ (シンプル、押し付けない)
  • キャラ 160px は 1 コース完了と同サイズ (最大級なので変えない)

📜 修了証カード仕様

  • Noto Serif JP を採用 (証書感、本文の Noto Sans JP と差別化)
  • ティール緑 2 重枠 (外枠 2px + 内枠 1px、4px インセット)
  • 角丸 6px シャープ (Nike エッセンス、紙の証書感)
  • 構成: タイトル → 氏名 → 本文 → 5 コース一覧 → 日付/発行者
  • 5 コース行は 各完了日付き (実装時 DB から自動入力)
  • スクショ保存推奨、html2canvas で PNG 出力も対応 ([画像を保存] ボタン)

🔗 SNS シェア 3 ボタン

  • [X でシェア]: Web Intent (`twitter.com/intent/tweet?text=...&url=...`) で簡単。テキスト雛形 + URL を自動挿入、X Card で修了証画像も出る
  • [画像を保存]: html2canvas で修了証カードを PNG ダウンロード → Instagram / LINE / 何にでも転載できる
  • [リンクをコピー]: 修了証の永続 URL (例: /graduation/u-1234) を clipboard へ
  • 3 つに絞った理由: Instagram は Web 直接シェア不可、画像 DL 経由が現実解

📊 サマリ数値 (5 コース通算、ポジティブ指標のみ)

  • 学習日数: 入塾日からの累計 (例: 189 日)
  • 視聴時間: 全レッスンの動画視聴時間合計 (例: 42.8 時間)
  • 完了レッスン: 「できた!」を押したレッスン数 (217 = 全レッスン数)
  • 否定的指標 (サボった日数 / 未完了数) は出さない

📝 メッセージのトーン

  • 見出し: 「全課程完了、おめでとうございます」 (「卒業」=サービスから離れる印象を避けるため不採用、「!」でなく「、」で静かに祝う)
  • 修了証本文: 「貴方は当塾の全 5 コース・22 章・217 レッスンを修了されました ここに証します」 (古典証書文体)
  • フッター 1 段目: 「ここまで来てくださって、ありがとうございました」 (受講生に感謝)
  • フッター 2 段目: 「私たち のりfitness も、これからも より良い学びをお届けし続けます / どうぞ、これからもよろしくお願いいたします」 (運営側からの継続宣言 + 関係性が続くことの示唆、終わりではない感)
  • 「天才」「最高」「すごい」は使わない
  • 受講生に「もっと頑張れ」とは言わない、運営側が「私たちも頑張り続けます」と宣言する形にしている

🎬 紙吹雪

  • 40 粒、5 色 (ティール / 黄 / オレンジ / 深緑 / ダークゴールド) + 円/四角混在
  • 3.3s〜4.6s で落下、ランダム遅延 0〜3.3s
  • infinite ループ (実装時に「最初 8 秒だけ」など制御可能)
  • course_complete の 2 倍、ただし派手すぎない密度に調整
